# Windows.UI.Xaml.Media.Imaging.BitmapSource.PixelWidth

## -description
Gets the width of the bitmap in pixels.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Imaging.BitmapSource.PixelWidth](/windows/winui/api/microsoft.ui.xaml.media.imaging.bitmapsource.pixelwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The width of the bitmap in pixels.
