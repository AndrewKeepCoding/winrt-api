# Windows.UI.Xaml.Media.Imaging

## -description

Provides types related to imaging and obtaining resources and source files for bitmap images.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Imaging](/windows/winui/api/microsoft.ui.xaml.media.imaging) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
