# Windows.UI.Xaml.Media.Imaging.BitmapSource.PixelWidthProperty

## -description
Identifies the [PixelWidth](bitmapsource_pixelwidth.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Imaging.BitmapSource.PixelWidthProperty](/windows/winui/api/microsoft.ui.xaml.media.imaging.bitmapsource.pixelwidthproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [PixelWidth](bitmapsource_pixelwidth.md) dependency property.
