# Windows.UI.Xaml.Documents.Glyphs.FontRenderingEmSize

## -description
Gets or sets the em size used for rendering the glyphs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Glyphs.FontRenderingEmSize](/windows/winui/api/microsoft.ui.xaml.documents.glyphs.fontrenderingemsize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
