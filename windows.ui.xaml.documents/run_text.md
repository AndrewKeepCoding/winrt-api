# Windows.UI.Xaml.Documents.Run.Text

## -description
Gets or sets the text contents of the [Run](run.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Run.Text](/windows/winui/api/microsoft.ui.xaml.documents.run.text) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<Run Text="text"/>
- or -
<Run ...>text</Run>
```


## -property-value
A string that specifies the text contents of the [Run](run.md). The default is [String.Empty](/dotnet/api/system.string.empty?view=dotnet-uwp-10.0&preserve-view=true).
