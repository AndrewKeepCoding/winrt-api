# Windows.UI.Xaml.Documents.Typography.SetContextualSwashes

## -description
Sets the value of the [ContextualSwashes](typography_contextualswashes.md) attached property for a specified dependency object.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Typography.SetContextualSwashes](/windows/winui/api/microsoft.ui.xaml.documents.typography.setcontextualswashes) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param element
The dependency object for which to set the value of the [ContextualSwashes](typography_contextualswashes.md) property.

### -param value
The new value to set the property to.
