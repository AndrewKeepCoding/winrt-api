# Windows.UI.Xaml.Documents.BlockCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.BlockCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.documents.blockcollection.replaceall).

## -parameters
### -param items
The new collection items.
