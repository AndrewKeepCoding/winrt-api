# Windows.UI.Xaml.Documents.TextElement.AccessKey

## -description
Gets or sets the access key for this element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.TextElement.AccessKey](/windows/winui/api/microsoft.ui.xaml.documents.textelement.accesskey) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The access key for this element.
