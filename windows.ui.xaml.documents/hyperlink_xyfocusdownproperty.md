# Windows.UI.Xaml.Documents.Hyperlink.XYFocusDownProperty

## -description
Identifies the [XYFocusDown](hyperlink_xyfocusdown.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Hyperlink.XYFocusDownProperty](/windows/winui/api/microsoft.ui.xaml.documents.hyperlink.xyfocusdownproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [XYFocusDown](hyperlink_xyfocusdown.md) dependency property.
