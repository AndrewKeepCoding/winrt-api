# Windows.UI.Xaml.Documents.Glyphs.UnicodeStringProperty

## -description
Identifies the [UnicodeString](glyphs_unicodestring.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Glyphs.UnicodeStringProperty](/windows/winui/api/microsoft.ui.xaml.documents.glyphs.unicodestringproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [UnicodeString](glyphs_unicodestring.md) dependency property.
