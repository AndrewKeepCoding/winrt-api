# Windows.UI.Xaml.Documents.TextHighlighter.Background

## -description

Gets or sets the brush used to highlight the text.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.TextHighlighter.Background](/windows/winui/api/microsoft.ui.xaml.documents.texthighlighter.background) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The brush used to highlight the text.
