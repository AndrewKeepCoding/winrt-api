# Windows.UI.Xaml.Documents.Hyperlink.XYFocusUpProperty

## -description
Identifies the [XYFocusUp](hyperlink_xyfocusup.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Hyperlink.XYFocusUpProperty](/windows/winui/api/microsoft.ui.xaml.documents.hyperlink.xyfocusupproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [XYFocusUp](hyperlink_xyfocusup.md) dependency property.
