# Windows.UI.Xaml.Documents.TextElement.AccessKeyDisplayRequested

## -description
Occurs when the access key sequence is started to notify controls that they should show access key visuals.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.TextElement.AccessKeyDisplayRequested](/windows/winui/api/microsoft.ui.xaml.documents.textelement.accesskeydisplayrequested) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks
This event is not raised if the element is under an element with its [Visibility](../windows.ui.xaml/uielement_visibility.md) property set to **Collapsed**.
