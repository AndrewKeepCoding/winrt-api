# Windows.UI.Xaml.Documents.Block.LineHeightProperty

## -description
Identifies the [LineHeight](block_lineheight.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Block.LineHeightProperty](/windows/winui/api/microsoft.ui.xaml.documents.block.lineheightproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [LineHeight](block_lineheight.md) dependency property.
