# Windows.UI.Xaml.Documents.Glyphs.OriginX

## -description
Gets or sets the x origin for the glyphs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Documents.Glyphs.OriginX](/windows/winui/api/microsoft.ui.xaml.documents.glyphs.originx) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
