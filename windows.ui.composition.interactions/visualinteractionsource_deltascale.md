# Windows.UI.Composition.Interactions.VisualInteractionSource.DeltaScale

## -description
The amount of scale change to be applied during the upcoming frame. Multiplicative. Defaults to 1. Readable only via expression/animation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Interactions.VisualInteractionSource.DeltaScale](/windows/winui/api/microsoft.ui.composition.interactions.visualinteractionsource.deltascale) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The change in scale of the visual interaction source.
