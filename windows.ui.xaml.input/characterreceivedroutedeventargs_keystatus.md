# Windows.UI.Xaml.Input.CharacterReceivedRoutedEventArgs.KeyStatus

## -description
Gets the status of the physical key that raised the character-received event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.CharacterReceivedRoutedEventArgs.KeyStatus](/windows/winui/api/microsoft.ui.xaml.input.characterreceivedroutedeventargs.keystatus) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The status of the key that was pressed.
