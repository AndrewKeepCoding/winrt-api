# Windows.UI.Xaml.Input.ManipulationPivot

## -description
Specifies how a rotation occurs with one point of user input.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationPivot](/windows/winui/api/microsoft.ui.xaml.input.manipulationpivot) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
