# Windows.UI.Xaml.Input.KeyboardAccelerator.IsEnabledProperty

## -description
Identifies the [IsEnabled](keyboardaccelerator_isenabled.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.KeyboardAccelerator.IsEnabledProperty](/windows/winui/api/microsoft.ui.xaml.input.keyboardaccelerator.isenabledproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsEnabled](keyboardaccelerator_isenabled.md) dependency property.
