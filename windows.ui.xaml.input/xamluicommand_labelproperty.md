# Windows.UI.Xaml.Input.XamlUICommand.LabelProperty

## -description

Identifies the [Label](xamluicommand_label.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.XamlUICommand.LabelProperty](/windows/winui/api/microsoft.ui.xaml.input.xamluicommand.labelproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the [Label](xamluicommand_label.md) dependency property.
