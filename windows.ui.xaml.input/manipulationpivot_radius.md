# Windows.UI.Xaml.Input.ManipulationPivot.Radius

## -description
Gets or sets the effective radius of rotation for rotation manipulations.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationPivot.Radius](/windows/winui/api/microsoft.ui.xaml.input.manipulationpivot.radius) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
