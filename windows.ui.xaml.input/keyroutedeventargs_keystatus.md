# Windows.UI.Xaml.Input.KeyRoutedEventArgs.KeyStatus

## -description
Gets a structure value that reports various system-detected characteristics of the key press, including repeat count and menu status.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.KeyRoutedEventArgs.KeyStatus](/windows/winui/api/microsoft.ui.xaml.input.keyroutedeventargs.keystatus) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A structure value with flags that report status.
