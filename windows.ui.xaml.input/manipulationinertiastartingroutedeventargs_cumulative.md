# Windows.UI.Xaml.Input.ManipulationInertiaStartingRoutedEventArgs.Cumulative

## -description

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationInertiaStartingRoutedEventArgs.Cumulative](/windows/winui/api/microsoft.ui.xaml.input.manipulationinertiastartingroutedeventargs.cumulative) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
