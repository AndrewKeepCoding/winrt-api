# Windows.UI.Xaml.Input.ManipulationStartingRoutedEventArgs.Mode

## -description
Gets or sets which types of manipulations are possible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationStartingRoutedEventArgs.Mode](/windows/winui/api/microsoft.ui.xaml.input.manipulationstartingroutedeventargs.mode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
One of the enumeration values. The default is **All**.
