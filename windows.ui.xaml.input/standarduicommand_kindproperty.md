# Windows.UI.Xaml.Input.StandardUICommand.KindProperty

## -description

Identifies the [Kind](standarduicommand_kind.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.StandardUICommand.KindProperty](/windows/winui/api/microsoft.ui.xaml.input.standarduicommand.kindproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the [Kind](standarduicommand_kind.md) dependency property.
