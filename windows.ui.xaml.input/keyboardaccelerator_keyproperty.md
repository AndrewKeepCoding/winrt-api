# Windows.UI.Xaml.Input.KeyboardAccelerator.KeyProperty

## -description
Identifies the [Key](keyboardaccelerator_key.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.KeyboardAccelerator.KeyProperty](/windows/winui/api/microsoft.ui.xaml.input.keyboardaccelerator.keyproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Key](keyboardaccelerator_key.md) dependency property.
