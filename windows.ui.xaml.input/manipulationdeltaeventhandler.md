# Windows.UI.Xaml.Input.ManipulationDeltaEventHandler

## -description
Represents the method that will handle [ManipulationDelta](../windows.ui.xaml/uielement_manipulationdelta.md) and related events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationDeltaEventHandler](/windows/winui/api/microsoft.ui.xaml.input.manipulationdeltaeventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The object where the handler is attached.

### -param e
Event data for the event.
