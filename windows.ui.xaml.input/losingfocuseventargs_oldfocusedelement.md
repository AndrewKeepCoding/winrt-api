# Windows.UI.Xaml.Input.LosingFocusEventArgs.OldFocusedElement

## -description
Gets the last focused object.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.LosingFocusEventArgs.OldFocusedElement](/windows/winui/api/microsoft.ui.xaml.input.losingfocuseventargs.oldfocusedelement) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The last focused object.
