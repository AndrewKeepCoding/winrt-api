# Windows.UI.Xaml.Input.StandardUICommand.Kind

## -description

Gets the platform command (with pre-defined properties such as icon, keyboard accelerator, and description) that can be used with a [StandardUICommand](standarduicommand.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.StandardUICommand.Kind](/windows/winui/api/microsoft.ui.xaml.input.standarduicommand.kind) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The pre-defined command.
