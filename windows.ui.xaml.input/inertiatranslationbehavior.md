# Windows.UI.Xaml.Input.InertiaTranslationBehavior

## -description
Controls deceleration on a translation manipulation during inertia.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.InertiaTranslationBehavior](/windows/winui/api/microsoft.ui.xaml.input.inertiatranslationbehavior) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
