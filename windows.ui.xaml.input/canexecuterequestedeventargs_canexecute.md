# Windows.UI.Xaml.Input.CanExecuteRequestedEventArgs.CanExecute

## -description

Gets or sets a value indicating whether the [ICommand](icommand.md) that raised this event is able to execute.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.CanExecuteRequestedEventArgs.CanExecute](/windows/winui/api/microsoft.ui.xaml.input.canexecuterequestedeventargs.canexecute) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

**true** if the ICommand is able to execute; otherwise, **false**.
