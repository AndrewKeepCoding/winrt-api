# Windows.UI.Xaml.Input.Pointer.PointerId

## -description
Gets the system-generated identifier for this pointer reference.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.Pointer.PointerId](/windows/winui/api/microsoft.ui.xaml.input.pointer.pointerid) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The system-generated identifier.
