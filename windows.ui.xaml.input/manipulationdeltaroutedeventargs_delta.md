# Windows.UI.Xaml.Input.ManipulationDeltaRoutedEventArgs.Delta

## -description
Gets the most recent changes of the current manipulation, as a [ManipulationDelta](manipulationdeltaroutedeventargs.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.ManipulationDeltaRoutedEventArgs.Delta](/windows/winui/api/microsoft.ui.xaml.input.manipulationdeltaroutedeventargs.delta) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The most recent changes of the current manipulation.
