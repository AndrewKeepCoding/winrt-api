# Windows.UI.Xaml.Input.XamlUICommand.Execute

## -description

Invokes the command.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Input.XamlUICommand.Execute](/windows/winui/api/microsoft.ui.xaml.input.xamluicommand.execute) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters

### -param parameter

Data used by the command. If the command does not require data, this object can be set to null.
