# Windows.UI.Text.FontWeights.Medium

## -description

Specifies a font weight value of 500.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.FontWeights.Medium](/windows/winui/api/microsoft.ui.text.fontweights.medium) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A [FontWeight](fontweight.md) with a Weight value of 500.
