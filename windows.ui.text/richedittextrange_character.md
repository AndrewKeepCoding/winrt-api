# Windows.UI.Text.RichEditTextRange.Character

## -description

Gets or sets the first character of the text range; that is, the character associated with the StartPosition property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextRange.Character](/windows/winui/api/microsoft.ui.text.richedittextrange.character) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The value of the first character in the text range.
