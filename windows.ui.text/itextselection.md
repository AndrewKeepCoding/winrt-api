# Windows.UI.Text.ITextSelection

## -description
Represents the currently selected text of a document.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextSelection](/windows/winui/api/microsoft.ui.text.itextselection) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks
### Interface inheritance

ITextSelection inherits [ITextRange](itextrange.md). Types that implement ITextSelection also implement the interface members of [ITextRange](itextrange.md).
