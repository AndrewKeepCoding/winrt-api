# Windows.UI.Text.ITextRange

## -description
Represents a span of continuous text in a document, and provides powerful editing and data-binding properties and methods that allow an app to select, examine, and change document text.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextRange](/windows/winui/api/microsoft.ui.text.itextrange) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
