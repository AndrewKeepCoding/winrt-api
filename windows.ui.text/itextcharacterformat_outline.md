# Windows.UI.Text.ITextCharacterFormat.Outline

## -description
Gets or sets whether characters are displayed as outlined characters.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.Outline](/windows/winui/api/microsoft.ui.text.itextcharacterformat.outline) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The outlined state.
