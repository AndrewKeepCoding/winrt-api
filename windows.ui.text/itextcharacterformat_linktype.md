# Windows.UI.Text.ITextCharacterFormat.LinkType

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.LinkType](/windows/winui/api/microsoft.ui.text.itextcharacterformat.linktype) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The link type.
