# Windows.UI.Text.ITextParagraphFormat.KeepTogether

## -description
Gets or sets whether page breaks are allowed in paragraphs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.KeepTogether](/windows/winui/api/microsoft.ui.text.itextparagraphformat.keeptogether) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that indicates whether page breaks are allowed.
