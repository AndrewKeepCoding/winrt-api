# Windows.UI.Text.ITextParagraphFormat.KeepWithNext

## -description
Gets or sets whether page breaks are allowed between paragraphs in a range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.KeepWithNext](/windows/winui/api/microsoft.ui.text.itextparagraphformat.keepwithnext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The value that indicates whether page breaks are allowed.
