# Windows.UI.Text.ITextCharacterFormat.FontStyle

## -description
Gets or sets the style of the font face, such as normal or italic.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.FontStyle](/windows/winui/api/microsoft.ui.text.itextcharacterformat.fontstyle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The font style.
