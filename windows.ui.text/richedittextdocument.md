# Windows.UI.Text.RichEditTextDocument

## -description
Represents a rich text document that can be loaded, saved, and edited.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextDocument](/windows/winui/api/microsoft.ui.text.richedittextdocument) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
