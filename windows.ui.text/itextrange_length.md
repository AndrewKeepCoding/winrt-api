# Windows.UI.Text.ITextRange.Length

## -description
Gets the count of characters in the text range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextRange.Length](/windows/winui/api/microsoft.ui.text.itextrange.length) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The count of characters.
