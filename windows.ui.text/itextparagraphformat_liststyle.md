# Windows.UI.Text.ITextParagraphFormat.ListStyle

## -description
Gets or sets the style used to mark the item paragraphs in a list.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.ListStyle](/windows/winui/api/microsoft.ui.text.itextparagraphformat.liststyle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The style used to mark the item paragraphs.
