# Windows.UI.Text.ITextParagraphFormat.RightToLeft

## -description
Gets or sets whether the paragraph uses right-to-left formatting.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.RightToLeft](/windows/winui/api/microsoft.ui.text.itextparagraphformat.righttoleft) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that indicates whether the paragraph uses right-to-left formatting.
