# Windows.UI.Text.FontWeights.Bold

## -description

Specifies a font weight value of 700.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.FontWeights.Bold](/windows/winui/api/microsoft.ui.text.fontweights.bold) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A [FontWeight](fontweight.md) with a Weight value of 700.
