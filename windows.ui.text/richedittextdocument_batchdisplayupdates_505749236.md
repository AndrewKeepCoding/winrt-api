# Windows.UI.Text.RichEditTextDocument.BatchDisplayUpdates


## -description

Increments an internal counter that controls whether text updates are displayed immediately or batched.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextDocument.BatchDisplayUpdates](/windows/winui/api/microsoft.ui.text.richedittextdocument.batchdisplayupdates) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns

The value of the internal counter.
