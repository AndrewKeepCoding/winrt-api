# Windows.UI.Text.TextConstants

## -description
Defines a set of constants that are used with various methods in the [Windows.UI.Text](windows_ui_text.md) namespace.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.TextConstants](/windows/winui/api/microsoft.ui.text.textconstants) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
