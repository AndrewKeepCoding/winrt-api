# Windows.UI.Text.ITextCharacterFormat.AllCaps

## -description
Gets or sets whether the characters are all uppercase.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.AllCaps](/windows/winui/api/microsoft.ui.text.itextcharacterformat.allcaps) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The uppercase state.
