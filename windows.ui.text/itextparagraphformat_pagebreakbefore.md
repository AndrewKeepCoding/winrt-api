# Windows.UI.Text.ITextParagraphFormat.PageBreakBefore

## -description
Gets or sets whether there is a page break before a paragraph.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.PageBreakBefore](/windows/winui/api/microsoft.ui.text.itextparagraphformat.pagebreakbefore) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that indicates whether there is page break.
