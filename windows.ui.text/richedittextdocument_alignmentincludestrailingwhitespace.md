# Windows.UI.Text.RichEditTextDocument.AlignmentIncludesTrailingWhitespace

## -description
Gets or sets a value that indicates whether trailing white space is taken into account when text is aligned.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextDocument.AlignmentIncludesTrailingWhitespace](/windows/winui/api/microsoft.ui.text.richedittextdocument.alignmentincludestrailingwhitespace) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if trailing whitespace is taken into account when text is aligned; **false** if trailing whitespace is ignored. The default is **false**.
