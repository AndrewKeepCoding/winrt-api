# Windows.UI.Text.ITextParagraphFormat.RightIndent

## -description
Gets or sets the right margin of a paragraph.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextParagraphFormat.RightIndent](/windows/winui/api/microsoft.ui.text.itextparagraphformat.rightindent) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The size of the right margin, in floating-point points.
