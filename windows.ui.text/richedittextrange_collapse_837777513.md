# Windows.UI.Text.RichEditTextRange.Collapse

## -description

Collapses the text range into a degenerate point at either the beginning or end of the range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextRange.Collapse](/windows/winui/api/microsoft.ui.text.richedittextrange.collapse) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param value

**true** to collapse at the start of the text range. **false** to collapse at the end of the range. The default is **true**.
