# Windows.UI.Text.RichEditTextRange.GetIndex

## -description

Retrieves the story index of the text unit (word, line, sentence, paragraph, and so on) at the starting character position of the text range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextRange.GetIndex](/windows/winui/api/microsoft.ui.text.richedittextrange.getindex) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param unit

The text unit that is indexed.

## -returns

The index value. The value is zero if unit does not exist.
