# Windows.UI.Text.RichEditTextDocument.CanRedo


## -description

Determines whether one or more redo operations exist.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.RichEditTextDocument.CanRedo](/windows/winui/api/microsoft.ui.text.richedittextdocument.canredo) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns

**true** if one or more redo operations exist; otherwise, **false**.
