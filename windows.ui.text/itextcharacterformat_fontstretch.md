# Windows.UI.Text.ITextCharacterFormat.FontStretch

## -description
Gets or sets the degree to which the font is stretched, compared to the normal aspect ratio of the font.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.FontStretch](/windows/winui/api/microsoft.ui.text.itextcharacterformat.fontstretch) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The degree to which the font is stretched.
