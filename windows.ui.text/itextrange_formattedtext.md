# Windows.UI.Text.ITextRange.FormattedText

## -description
Gets or sets an [ITextRange](itextrange.md) object with the formatted text of the specified range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextRange.FormattedText](/windows/winui/api/microsoft.ui.text.itextrange.formattedtext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The formatted text.
