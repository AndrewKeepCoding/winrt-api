# Windows.UI.Text.ITextCharacterFormat.SmallCaps

## -description
Gets or sets whether characters are in small capital letters.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.SmallCaps](/windows/winui/api/microsoft.ui.text.itextcharacterformat.smallcaps) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The small capitals state.
