# Windows.UI.Text.ITextCharacterFormat.LanguageTag

## -description
Gets or sets the Internet Engineering Task Force (IETF) language tag (BCP 47 standard) that identifies the language currently associated with the characters.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Text.ITextCharacterFormat.LanguageTag](/windows/winui/api/microsoft.ui.text.itextcharacterformat.languagetag) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The language tag.
