# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetItemType

## -description
Gets a string that describes what kind of item an element represents.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetItemType](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.getitemtype) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The kind of item.
