# Windows.UI.Xaml.Automation.Peers.RadioButtonAutomationPeer.Select

## -description
Clears any existing selection and then selects the current element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.RadioButtonAutomationPeer.Select](/windows/winui/api/microsoft.ui.xaml.automation.peers.radiobuttonautomationpeer.select) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
