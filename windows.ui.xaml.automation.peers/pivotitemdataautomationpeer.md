# Windows.UI.Xaml.Automation.Peers.PivotItemDataAutomationPeer

## -description
Exposes the data content of a [PivotItem](../windows.ui.xaml.controls/pivotitem.md) to Microsoft UI Automation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.PivotItemDataAutomationPeer](/windows/winui/api/microsoft.ui.xaml.automation.peers.pivotitemdataautomationpeer) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
