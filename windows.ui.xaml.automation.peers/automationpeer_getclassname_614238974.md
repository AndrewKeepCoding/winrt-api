# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetClassName

## -description
Gets a name that is used with [AutomationControlType](automationcontroltype.md), to differentiate the control that is represented by this [AutomationPeer](automationpeer.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetClassName](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.getclassname) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The class name.
