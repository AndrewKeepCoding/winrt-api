# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetHelpText

## -description
Gets text that describes the functionality of the control that is associated with the automation peer.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetHelpText](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.gethelptext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The help text.
