# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetAcceleratorKey

## -description
Gets the accelerator key combinations for the object that is associated with the UI Automation peer.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetAcceleratorKey](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.getacceleratorkey) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The accelerator key.
