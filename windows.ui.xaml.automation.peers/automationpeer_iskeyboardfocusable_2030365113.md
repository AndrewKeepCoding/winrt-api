# Windows.UI.Xaml.Automation.Peers.AutomationPeer.IsKeyboardFocusable

## -description
Gets a value that indicates whether the element can accept keyboard focus.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.IsKeyboardFocusable](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.iskeyboardfocusable) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
**true** if the element can accept keyboard focus; otherwise, **false**.
