# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetAnnotationsCore

## -description
Provides the behavior of the peer when a Microsoft UI Automation client calls [GetAnnotations](automationpeer_getannotations_676104854.md) or an equivalent Microsoft UI Automation client API.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetAnnotationsCore](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.getannotationscore) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The behavior of the peer.
