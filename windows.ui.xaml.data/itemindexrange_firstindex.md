# Windows.UI.Xaml.Data.ItemIndexRange.FirstIndex

## -description
Gets the index of the first item in the instance of the [ItemIndexRange](itemindexrange.md) class.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ItemIndexRange.FirstIndex](/windows/winui/api/microsoft.ui.xaml.data.itemindexrange.firstindex) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The index of the first item in the instance of the [ItemIndexRange](itemindexrange.md) class.
