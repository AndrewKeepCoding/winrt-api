# Windows.UI.Xaml.Data.ICollectionView.CurrentChanged

## -description
When implementing this interface, fire this event after the current item has been changed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ICollectionView.CurrentChanged](/windows/winui/api/microsoft.ui.xaml.data.icollectionview.currentchanged) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks
Don't implement this interface for Microsoft .NET. See Remarks in [ICollectionView](icollectionview.md).
