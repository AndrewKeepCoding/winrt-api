# Windows.UI.Xaml.Data.BindingExpression.DataItem

## -description
Gets the binding source object that this [BindingExpression](bindingexpression.md) uses.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.BindingExpression.DataItem](/windows/winui/api/microsoft.ui.xaml.data.bindingexpression.dataitem) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The binding source object that this [BindingExpression](bindingexpression.md) uses.
