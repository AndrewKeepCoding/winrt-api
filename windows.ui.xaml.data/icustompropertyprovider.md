# Windows.UI.Xaml.Data.ICustomPropertyProvider

## -description
Provides lookup service for [ICustomProperty](icustomproperty.md) support. This interface is implemented by objects so that their custom defined properties can be used as run-time binding sources.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ICustomPropertyProvider](/windows/winui/api/microsoft.ui.xaml.data.icustompropertyprovider) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
