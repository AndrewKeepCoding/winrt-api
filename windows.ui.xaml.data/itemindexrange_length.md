# Windows.UI.Xaml.Data.ItemIndexRange.Length

## -description
Gets the number of items in the instance of the [ItemIndexRange](itemindexrange.md) class.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ItemIndexRange.Length](/windows/winui/api/microsoft.ui.xaml.data.itemindexrange.length) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The number of items in the instance of the [ItemIndexRange](itemindexrange.md) class.
