# Windows.UI.Xaml.Data.ICustomProperty.SetValue

## -description
Sets the custom property value on a specified instance.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ICustomProperty.SetValue](/windows/winui/api/microsoft.ui.xaml.data.icustomproperty.setvalue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param target
The owner instance.

### -param value
The value to set.
