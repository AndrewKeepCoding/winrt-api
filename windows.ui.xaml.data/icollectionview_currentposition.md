# Windows.UI.Xaml.Data.ICollectionView.CurrentPosition

## -description
Gets the ordinal position of the [CurrentItem](icollectionview_currentitem.md) within the view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ICollectionView.CurrentPosition](/windows/winui/api/microsoft.ui.xaml.data.icollectionview.currentposition) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The ordinal position of the [CurrentItem](icollectionview_currentitem.md) within the view.
