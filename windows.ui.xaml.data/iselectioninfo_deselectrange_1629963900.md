# Windows.UI.Xaml.Data.ISelectionInfo.DeselectRange

## -description
Marks the items in the data source specified by *itemIndexRange* as not selected in the list control.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Data.ISelectionInfo.DeselectRange](/windows/winui/api/microsoft.ui.xaml.data.iselectioninfo.deselectrange) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param itemIndexRange
A range of items in the data source.
