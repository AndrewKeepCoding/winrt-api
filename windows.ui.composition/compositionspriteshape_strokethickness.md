# Windows.UI.Composition.CompositionSpriteShape.StrokeThickness

## -description

Gets or sets the width of the shape outline.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionSpriteShape.StrokeThickness](/windows/winui/api/microsoft.ui.composition.compositionspriteshape.strokethickness) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The width of the shape outline, in pixels. The default value is 0.
