# Windows.UI.Composition.Visual.BackfaceVisibility

## -description
Specifies whether the back face of the visual should be visible during a 3D transform.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Visual.BackfaceVisibility](/windows/winui/api/microsoft.ui.composition.visual.backfacevisibility) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
Whether the back face of the visual should be visible during a 3D transform.
