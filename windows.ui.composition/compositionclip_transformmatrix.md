# Windows.UI.Composition.CompositionClip.TransformMatrix

## -description
The 3x2 transformation matrix to apply to the clip.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionClip.TransformMatrix](/windows/winui/api/microsoft.ui.composition.compositionclip.transformmatrix) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The transformation matrix to apply to the clip. By default, the TransformMatrix is set to:

```

[1 0
 0 1
 0 0]             
            
```
