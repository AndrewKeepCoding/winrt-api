# Windows.UI.Composition.AnimationPropertyInfo.AccessMode

## -description

Gets or sets a value that specifies how an animated property can be accessed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.AnimationPropertyInfo.AccessMode](/windows/winui/api/microsoft.ui.composition.animationpropertyinfo.accessmode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A value of the enumeration that specifies how an animated property can be accessed.
