# Windows.UI.Composition.CompositionShape.RotationAngle

## -description

Gets or sets the rotation angle of the shape in radians.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionShape.RotationAngle](/windows/winui/api/microsoft.ui.composition.compositionshape.rotationangle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The rotation angle of the shape in radians.
