# Windows.UI.Composition.Compositor.CreatePointLight

## -description
Creates an instance of [PointLight](pointlight.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreatePointLight](/windows/winui/api/microsoft.ui.composition.compositor.createpointlight) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [PointLight](pointlight.md) object.
