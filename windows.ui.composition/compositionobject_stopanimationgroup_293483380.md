# Windows.UI.Composition.CompositionObject.StopAnimationGroup

## -description
Stops an animation group.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionObject.StopAnimationGroup](/windows/winui/api/microsoft.ui.composition.compositionobject.stopanimationgroup) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param value
The animation group to stop.
