# Windows.UI.Composition.CompositionLineGeometry.Start

## -description

Gets or sets the starting point of the line.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionLineGeometry.Start](/windows/winui/api/microsoft.ui.composition.compositionlinegeometry.start) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The starting point of the line.
