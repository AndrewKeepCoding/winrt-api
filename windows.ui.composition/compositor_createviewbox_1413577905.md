# Windows.UI.Composition.Compositor.CreateViewBox

## -description

Creates an instance of [CompositionViewBox](compositionviewbox.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateViewBox](/windows/winui/api/microsoft.ui.composition.compositor.createviewbox) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns

Returns the created [CompositionViewBox](compositionviewbox.md) object.
