# Windows.UI.Composition.AnimationControllerProgressBehavior

## -description

Defines constants that specify how the [AnimationController.Progress](animationcontroller_progress.md) value is determined.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.AnimationControllerProgressBehavior](/windows/winui/api/microsoft.ui.composition.animationcontrollerprogressbehavior) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -enum-fields
### -field IncludesDelayTime:1

The progress value includes delay time.

### -field Default:0

The progress value does not include delay time.
