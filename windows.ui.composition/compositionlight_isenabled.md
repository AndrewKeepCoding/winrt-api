# Windows.UI.Composition.CompositionLight.IsEnabled

## -description

Gets or sets a value that determines whether the composition light is on.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionLight.IsEnabled](/windows/winui/api/microsoft.ui.composition.compositionlight.isenabled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

**true** if the light is on; otherwise, **false**. The default is **true**.
