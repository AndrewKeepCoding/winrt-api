# Windows.UI.Composition.Compositor.CreatePropertySet

## -description
Creates an instance of [CompositionPropertySet](compositionpropertyset.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreatePropertySet](/windows/winui/api/microsoft.ui.composition.compositor.createpropertyset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [CompositionPropertySet](compositionpropertyset.md) object.
