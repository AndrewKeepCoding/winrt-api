# AnimationDirection

## -description
Specifies the play direction of an animation.

Equivalent **WinUI 2 API for UWP**: [AnimationDirection](/windows/winui/api/microsoft.ui.composition.animationdirection) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -enum-fields
### -field Normal:0
Play the animation from start to end.

### -field Reverse:1
Play the animation in reverse from end to start.

### -field Alternate:2
Alternate between normal and reverse playback.

### -field AlternateReverse:3
Alternate between reverse and normal playback.
