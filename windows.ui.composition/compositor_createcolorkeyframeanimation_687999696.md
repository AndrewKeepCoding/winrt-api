# Windows.UI.Composition.Compositor.CreateColorKeyFrameAnimation

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateColorKeyFrameAnimation](/windows/winui/api/microsoft.ui.composition.compositor.createcolorkeyframeanimation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [ColorKeyFrameAnimation](colorkeyframeanimation.md) object.
