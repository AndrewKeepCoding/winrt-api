# Windows.UI.Composition.CompositionRoundedRectangleGeometry.Size

## -description

Gets or sets the height and width of the rectangle.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionRoundedRectangleGeometry.Size](/windows/winui/api/microsoft.ui.composition.compositionroundedrectanglegeometry.size) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The height and width of the rectangle.
