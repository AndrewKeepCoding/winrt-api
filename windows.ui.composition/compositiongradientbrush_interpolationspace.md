# Windows.UI.Composition.CompositionGradientBrush.InterpolationSpace

## -description

Gets or sets a value that specifies how the gradient's colors are interpolated.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionGradientBrush.InterpolationSpace](/windows/winui/api/microsoft.ui.composition.compositiongradientbrush.interpolationspace) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A value of the enumeration that specifies how the gradient's colors are interpolated.
