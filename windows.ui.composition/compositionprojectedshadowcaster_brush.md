# Windows.UI.Composition.CompositionProjectedShadowCaster.Brush

## -description

Gets or sets the brush used to draw the shadow.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionProjectedShadowCaster.Brush](/windows/winui/api/microsoft.ui.composition.compositionprojectedshadowcaster.brush) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The brush used to draw the shadow.

## -remarks

The brush must be a [CompositionColorBrush](compositioncolorbrush.md).
