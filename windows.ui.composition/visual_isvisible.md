# Windows.UI.Composition.Visual.IsVisible

## -description
Indicates whether the visual and its entire subtree of child visuals is visible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Visual.IsVisible](/windows/winui/api/microsoft.ui.composition.visual.isvisible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
Indicates whether the visual and its entire subtree of child visuals is visible.
