# Windows.UI.Composition.VisualCollection.RemoveAll

## -description
Removes all of the visual instances from the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.VisualCollection.RemoveAll](/windows/winui/api/microsoft.ui.composition.visualcollection.removeall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
