# Windows.UI.Composition.SpotLight.OuterConeIntensity

## -description

Gets or sets the intensity of the light in the spotlight's outer cone.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.SpotLight.OuterConeIntensity](/windows/winui/api/microsoft.ui.composition.spotlight.outerconeintensity) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The intensity of the light.
