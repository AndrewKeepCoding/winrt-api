# Windows.UI.Composition.CompositionVisualSurface

## -description

Represents a visual tree as an ICompositionSurface that can be used to paint a Visual using a CompositionBrush.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionVisualSurface](/windows/winui/api/microsoft.ui.composition.compositionvisualsurface) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
