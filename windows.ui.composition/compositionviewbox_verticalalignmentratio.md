# Windows.UI.Composition.CompositionViewBox.VerticalAlignmentRatio

## -description

Gets or sets the vertical alignment ratio of the view box.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionViewBox.VerticalAlignmentRatio](/windows/winui/api/microsoft.ui.composition.compositionviewbox.verticalalignmentratio) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The vertical alignment ratio of the view box.

The default value is set to 0.5.
