# Windows.UI.Composition.CompositionLinearGradientBrush.EndPoint

## -description

Gets or sets the ending two-dimensional coordinates of the linear gradient.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionLinearGradientBrush.EndPoint](/windows/winui/api/microsoft.ui.composition.compositionlineargradientbrush.endpoint) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The ending two-dimensional coordinates of the linear gradient.
