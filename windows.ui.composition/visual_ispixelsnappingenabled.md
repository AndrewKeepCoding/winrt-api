# Windows.UI.Composition.Visual.IsPixelSnappingEnabled

<!--

## -description

Gets or sets a value that indicates whether the visual aligns with pixel boundaries.

## -property-value

`true` if the visual aligns with pixel boundaries; otherwise, `false`.
