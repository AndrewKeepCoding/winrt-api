# Windows.UI.Composition.CompositionObject.Close

## -description
Closes the CompositionObject and releases system resources.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionObject.Close](/windows/winui/api/microsoft.ui.composition.compositionobject.close) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
