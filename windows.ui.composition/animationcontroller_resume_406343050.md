# Windows.UI.Composition.AnimationController.Resume

## -description

Starts playback of an animation that was previously paused.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.AnimationController.Resume](/windows/winui/api/microsoft.ui.composition.animationcontroller.resume) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
