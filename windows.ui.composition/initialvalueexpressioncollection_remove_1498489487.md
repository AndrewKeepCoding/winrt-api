# Windows.UI.Composition.InitialValueExpressionCollection.Remove

## -description

Removes the specified item from the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.InitialValueExpressionCollection.Remove](/windows/winui/api/microsoft.ui.composition.initialvalueexpressioncollection.remove) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters

### -param key

The key that identifies the item to remove.
