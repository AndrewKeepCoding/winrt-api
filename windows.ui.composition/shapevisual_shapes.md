# Windows.UI.Composition.ShapeVisual.Shapes

## -description

Gets the collection of CompositionShapes that this shape visual tree is composed of.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.ShapeVisual.Shapes](/windows/winui/api/microsoft.ui.composition.shapevisual.shapes) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The collection of CompositionShapes that this shape visual tree is composed of.
