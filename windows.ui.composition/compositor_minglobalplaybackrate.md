# Windows.UI.Composition.Compositor.MinGlobalPlaybackRate

## -description

Gets the minimum allowed playback rate for all KeyFrame animations created by this compositor.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.MinGlobalPlaybackRate](/windows/winui/api/microsoft.ui.composition.compositor.minglobalplaybackrate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The minimum allowed playback rate.
