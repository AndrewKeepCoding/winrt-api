# Windows.UI.Composition.CompositionColorGradientStop

## -description

Describes the location and color of a transition point in a gradient.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionColorGradientStop](/windows/winui/api/microsoft.ui.composition.compositioncolorgradientstop) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
