# Windows.UI.Composition.SpotLight.OuterConeColor

## -description
The color of the spotlight's outer cone. Animatable.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.SpotLight.OuterConeColor](/windows/winui/api/microsoft.ui.composition.spotlight.outerconecolor) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The color of the spotlight's outer cone.
