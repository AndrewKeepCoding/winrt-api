# Windows.UI.Composition.CompositionDrawingSurface.Resize

## -description
Resizes the drawing surface to the specified size.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionDrawingSurface.Resize](/windows/winui/api/microsoft.ui.composition.compositiondrawingsurface.resize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters

### -param sizePixels
The new size in pixels for the drawing surface.
