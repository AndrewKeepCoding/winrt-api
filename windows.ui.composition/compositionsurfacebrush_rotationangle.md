# Windows.UI.Composition.CompositionSurfaceBrush.RotationAngle

## -description
The rotation angle, in radians, of the brush.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionSurfaceBrush.RotationAngle](/windows/winui/api/microsoft.ui.composition.compositionsurfacebrush.rotationangle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The rotation angle, in radians, of the brush.
