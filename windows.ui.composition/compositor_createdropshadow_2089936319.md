# Windows.UI.Composition.Compositor.CreateDropShadow

## -description
Creates an instance of [DropShadow](dropshadow.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateDropShadow](/windows/winui/api/microsoft.ui.composition.compositor.createdropshadow) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [DropShadow](dropshadow.md) object.
