# Windows.UI.Composition.CompositionStrokeDashArray.ReplaceAll

## -description

Initially clears the collection, then inserts the provided array as new items.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionStrokeDashArray.ReplaceAll](/windows/winui/api/microsoft.ui.composition.compositionstrokedasharray.replaceall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param items

The new collection items.
