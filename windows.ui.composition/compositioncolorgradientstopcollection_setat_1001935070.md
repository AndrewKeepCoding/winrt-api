# Windows.UI.Composition.CompositionColorGradientStopCollection.SetAt

## -description

Sets the value at the specified index to the [CompositionColorGradientStop](compositioncolorgradientstop.md) value specified.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionColorGradientStopCollection.SetAt](/windows/winui/api/microsoft.ui.composition.compositioncolorgradientstopcollection.setat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index

The index at which to set the value.

### -param value

The value to set.
