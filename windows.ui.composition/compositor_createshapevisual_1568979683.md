# Windows.UI.Composition.Compositor.CreateShapeVisual

## -description

Creates an instance of [ShapeVisual](shapevisual.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateShapeVisual](/windows/winui/api/microsoft.ui.composition.compositor.createshapevisual) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns

Returns the created [ShapeVisual](shapevisual.md) object.
