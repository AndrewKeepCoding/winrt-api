# Windows.UI.Composition.CompositionLinearGradientBrush

## -description

Represents a brush that paints an area with a linear gradient.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionLinearGradientBrush](/windows/winui/api/microsoft.ui.composition.compositionlineargradientbrush) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
