# Windows.UI.Composition.CompositionStrokeDashArray.RemoveAtEnd

## -description

Removes the last item in the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionStrokeDashArray.RemoveAtEnd](/windows/winui/api/microsoft.ui.composition.compositionstrokedasharray.removeatend) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks

If successful, calling this method reduces the collection size by one.
