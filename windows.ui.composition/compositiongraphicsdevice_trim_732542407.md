# Windows.UI.Composition.CompositionGraphicsDevice.Trim

## -description

Trims any graphics memory allocated by the graphics device on the app's behalf.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionGraphicsDevice.Trim](/windows/winui/api/microsoft.ui.composition.compositiongraphicsdevice.trim) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
