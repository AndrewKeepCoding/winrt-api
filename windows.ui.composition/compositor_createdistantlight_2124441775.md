# Windows.UI.Composition.Compositor.CreateDistantLight

## -description
Creates an instance of [DistantLight](distantlight.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateDistantLight](/windows/winui/api/microsoft.ui.composition.compositor.createdistantlight) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [DistantLight](distantlight.md) object.
