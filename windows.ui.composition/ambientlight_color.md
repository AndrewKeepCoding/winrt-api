# Windows.UI.Composition.AmbientLight.Color

## -description
Color of the light. Animatable.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.AmbientLight.Color](/windows/winui/api/microsoft.ui.composition.ambientlight.color) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The color of the ambient light.
