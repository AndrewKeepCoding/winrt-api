# Windows.UI.Composition.Compositor.Close

## -description
Closes the Compositor object and releases system resources.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.Close](/windows/winui/api/microsoft.ui.composition.compositor.close) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
