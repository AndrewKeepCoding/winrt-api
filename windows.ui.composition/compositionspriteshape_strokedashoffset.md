# Windows.UI.Composition.CompositionSpriteShape.StrokeDashOffset

## -description

Gets or sets a value that specifies the distance within the dash pattern where a dash begins.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionSpriteShape.StrokeDashOffset](/windows/winui/api/microsoft.ui.composition.compositionspriteshape.strokedashoffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A value that represents the distance within the dash pattern where a dash begins. The default is 0.
