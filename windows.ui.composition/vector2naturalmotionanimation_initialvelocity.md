# Windows.UI.Composition.Vector2NaturalMotionAnimation.InitialVelocity

## -description

Gets or sets the velocity that the animation starts with.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Vector2NaturalMotionAnimation.InitialVelocity](/windows/winui/api/microsoft.ui.composition.vector2naturalmotionanimation.initialvelocity) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The velocity that the animation starts with. The default is 0.
