# Windows.UI.Composition.CompositionGeometricClip

## -description

Represents a shape that clips a portion of a visual. The visible portion of the visual is a shape defined by a [CompositionGeometry](compositiongeometry.md). The portion of the visual outside the geometry is clipped.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionGeometricClip](/windows/winui/api/microsoft.ui.composition.compositiongeometricclip) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
