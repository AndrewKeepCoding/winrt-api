# Windows.UI.Composition.CompositionMipmapSurface.AlphaMode

## -description

Gets the alpha mode of the mipmap surface.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionMipmapSurface.AlphaMode](/windows/winui/api/microsoft.ui.composition.compositionmipmapsurface.alphamode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The alpha mode of the mipmap surface.
