# Windows.UI.Composition.CompositionColorBrush.Color

## -description
The color used to fill a [SpriteVisual](spritevisual.md). Animatable.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionColorBrush.Color](/windows/winui/api/microsoft.ui.composition.compositioncolorbrush.color) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The color used to fill a [SpriteVisual](spritevisual.md).
