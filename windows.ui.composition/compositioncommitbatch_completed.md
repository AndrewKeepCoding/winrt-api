# Windows.UI.Composition.CompositionCommitBatch.Completed

## -description
Event that is triggered when all objects in a [CompositionCommitBatch](compositioncommitbatch.md) have completed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionCommitBatch.Completed](/windows/winui/api/microsoft.ui.composition.compositioncommitbatch.completed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
