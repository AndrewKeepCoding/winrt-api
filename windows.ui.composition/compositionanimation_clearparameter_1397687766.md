# Windows.UI.Composition.CompositionAnimation.ClearParameter

## -description
Clears a parameter from the animation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionAnimation.ClearParameter](/windows/winui/api/microsoft.ui.composition.compositionanimation.clearparameter) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param key
The parameter to clear.
