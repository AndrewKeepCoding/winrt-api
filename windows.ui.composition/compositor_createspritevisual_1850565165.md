# Windows.UI.Composition.Compositor.CreateSpriteVisual

## -description
Creates an instance of [SpriteVisual](spritevisual.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateSpriteVisual](/windows/winui/api/microsoft.ui.composition.compositor.createspritevisual) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [SpriteVisual](spritevisual.md) object.
