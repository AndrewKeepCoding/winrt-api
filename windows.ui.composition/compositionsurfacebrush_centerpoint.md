# Windows.UI.Composition.CompositionSurfaceBrush.CenterPoint

## -description
The point about which the brush is rotated and scaled.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionSurfaceBrush.CenterPoint](/windows/winui/api/microsoft.ui.composition.compositionsurfacebrush.centerpoint) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The point about which the brush is rotated and scaled.
