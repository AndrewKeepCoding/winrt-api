# Windows.UI.Composition.CompositionEllipseGeometry.Center

## -description

Gets or sets the center point of the ellipse.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionEllipseGeometry.Center](/windows/winui/api/microsoft.ui.composition.compositionellipsegeometry.center) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The center point of the ellipse.
