# Windows.UI.Composition.NaturalMotionAnimation

## -description

Represents a physics-based animation that uses forces to dynamically move content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.NaturalMotionAnimation](/windows/winui/api/microsoft.ui.composition.naturalmotionanimation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
