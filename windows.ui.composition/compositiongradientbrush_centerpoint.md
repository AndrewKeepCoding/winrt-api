# Windows.UI.Composition.CompositionGradientBrush.CenterPoint

## -description

Gets or sets the point about which the brush is rotated and scaled.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionGradientBrush.CenterPoint](/windows/winui/api/microsoft.ui.composition.compositiongradientbrush.centerpoint) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The point about which the brush is rotated and scaled.
