# Windows.UI.Composition.CompositionGraphicsDevice.CreateDrawingSurface2

## -description
Creates an instance of CompositionDrawingSurface.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionGraphicsDevice.CreateDrawingSurface2](/windows/winui/api/microsoft.ui.composition.compositiongraphicsdevice.createdrawingsurface2) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters

### -param sizePixels
The size of the drawing surface in pixels.

### -param pixelFormat
The pixel format of the drawing surface.

### -param alphaMode
The alpha mode of the drawing surface.

## -returns
Returns the created CompositionDrawingSurface.
