# Windows.UI.Composition.CompositionShadow

## -description
Base class for shadows that can be applied to a [SpriteVisual](spritevisual.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionShadow](/windows/winui/api/microsoft.ui.composition.compositionshadow) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
