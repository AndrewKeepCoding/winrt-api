# Windows.UI.Composition.CompositionVisualSurface.SourceSize

## -description

Gets or sets the the height and width of the part of the visual surface used for rendering.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionVisualSurface.SourceSize](/windows/winui/api/microsoft.ui.composition.compositionvisualsurface.sourcesize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The height and width of the part of the visual surface used for rendering.
