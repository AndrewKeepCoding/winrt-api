# Windows.UI.Composition.Compositor.CreateAnimationGroup

## -description
Creates an instance of [CompositionAnimationGroup](compositionanimationgroup.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Compositor.CreateAnimationGroup](/windows/winui/api/microsoft.ui.composition.compositor.createanimationgroup) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
Returns the created [CompositionAnimationGroup](compositionanimationgroup.md) object.
