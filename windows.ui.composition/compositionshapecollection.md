# Windows.UI.Composition.CompositionShapeCollection

## -description

Represents a collection of [CompositionShape](compositionshape.md) objects that can be individually accessed by index.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionShapeCollection](/windows/winui/api/microsoft.ui.composition.compositionshapecollection) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
