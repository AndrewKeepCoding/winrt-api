# Windows.UI.Composition.CompositionSurfaceBrush.Offset

## -description
The offset of the brush relative to its [SpriteVisual](spritevisual.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionSurfaceBrush.Offset](/windows/winui/api/microsoft.ui.composition.compositionsurfacebrush.offset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The offset of the brush relative to its [SpriteVisual](spritevisual.md).
