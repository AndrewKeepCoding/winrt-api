# Windows.UI.Composition.CompositionScopedBatch.Completed

## -description
Event triggered once all animations and effects in the [CompositionScopedBatch](compositionscopedbatch.md) have completed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.CompositionScopedBatch.Completed](/windows/winui/api/microsoft.ui.composition.compositionscopedbatch.completed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
