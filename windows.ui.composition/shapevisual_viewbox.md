# Windows.UI.Composition.ShapeVisual.ViewBox

## -description

Gets or sets the CompositionViewBox that maps shape visual tree coordinates onto the visual.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.ShapeVisual.ViewBox](/windows/winui/api/microsoft.ui.composition.shapevisual.viewbox) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The CompositionViewBox that maps shape visual tree coordinates onto the visual.
