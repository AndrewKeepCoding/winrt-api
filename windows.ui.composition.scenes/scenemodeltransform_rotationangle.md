# Windows.UI.Composition.Scenes.SceneModelTransform.RotationAngle

## -description

Gets or sets the rotation angle applied to the node, in radians.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneModelTransform.RotationAngle](/windows/winui/api/microsoft.ui.composition.scenes.scenemodeltransform.rotationangle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The rotation angle applied to the node, in radians.
