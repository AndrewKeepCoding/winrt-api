# Windows.UI.Composition.Scenes.SceneMeshRendererComponent.Material

## -description

Gets or sets the material that describes how the mesh surface is rendered.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneMeshRendererComponent.Material](/windows/winui/api/microsoft.ui.composition.scenes.scenemeshrenderercomponent.material) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The material that describes how the mesh surface is rendered.
