# Windows.UI.Composition.Scenes.SceneModelTransform.Scale

## -description

Gets or sets the relative size of the node on the x, y, and z axes.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneModelTransform.Scale](/windows/winui/api/microsoft.ui.composition.scenes.scenemodeltransform.scale) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A vector that describes the relative size of the node on the x, y, and z axes.
