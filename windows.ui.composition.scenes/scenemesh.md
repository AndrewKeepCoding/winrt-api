# Windows.UI.Composition.Scenes.SceneMesh

## -description

Represents a mesh (the list of vertices that describe the polygons that make up a scene).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneMesh](/windows/winui/api/microsoft.ui.composition.scenes.scenemesh) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
