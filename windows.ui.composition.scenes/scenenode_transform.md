# Windows.UI.Composition.Scenes.SceneNode.Transform

## -description

Gets the transform that describes the position, rotation, and scale of this node.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneNode.Transform](/windows/winui/api/microsoft.ui.composition.scenes.scenenode.transform) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The transform that describes the position, rotation, and scale of this node.
