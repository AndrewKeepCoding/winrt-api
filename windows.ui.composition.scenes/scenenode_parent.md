# Windows.UI.Composition.Scenes.SceneNode.Parent

## -description

Gets the node that this node is a child of.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Scenes.SceneNode.Parent](/windows/winui/api/microsoft.ui.composition.scenes.scenenode.parent) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The node that this node is a child of.
