# Windows.UI.Xaml.Resources.CustomXamlResourceLoader.Current

## -description

Returns the active [CustomXamlResourceLoader](customxamlresourceloader.md) instance.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Resources.CustomXamlResourceLoader.Current](/windows/winui/api/microsoft.ui.xaml.resources.customxamlresourceloader.current) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The active [CustomXamlResourceLoader](customxamlresourceloader.md) instance.
