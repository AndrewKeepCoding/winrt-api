# Windows.UI.Xaml.Resources

## -description

Contains [CustomXamlResourceLoader](customxamlresourceloader.md), which enables loading of custom resource dictionaries as sources.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Resources](/windows/winui/api/microsoft.ui.xaml.resources) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
