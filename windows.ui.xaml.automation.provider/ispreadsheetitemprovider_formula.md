# Windows.UI.Xaml.Automation.Provider.ISpreadsheetItemProvider.Formula

## -description
Gets the formula for this spreadsheet cell, as a string.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ISpreadsheetItemProvider.Formula](/windows/winui/api/microsoft.ui.xaml.automation.provider.ispreadsheetitemprovider.formula) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The formula for this cell, as a string.
