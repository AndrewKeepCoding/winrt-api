# Windows.UI.Xaml.Automation.Provider.ITextProvider.SupportedTextSelection

## -description
Gets a value that specifies whether a text provider supports selection, and if it does, the type of selection that is supported.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ITextProvider.SupportedTextSelection](/windows/winui/api/microsoft.ui.xaml.automation.provider.itextprovider.supportedtextselection) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value of [SupportedTextSelection](../windows.ui.xaml.automation/supportedtextselection.md).
