# Windows.UI.Xaml.Automation.Provider.IWindowProvider.Maximizable

## -description
Gets a value that specifies whether the window can be maximized.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.IWindowProvider.Maximizable](/windows/winui/api/microsoft.ui.xaml.automation.provider.iwindowprovider.maximizable) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the window can be maximized; otherwise, **false**.
