# Windows.UI.Xaml.Automation.Provider.IStylesProvider

## -description
Provides access to the visual styles associated with the content of a document.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.IStylesProvider](/windows/winui/api/microsoft.ui.xaml.automation.provider.istylesprovider) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
