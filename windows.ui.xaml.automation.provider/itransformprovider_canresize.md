# Windows.UI.Xaml.Automation.Provider.ITransformProvider.CanResize

## -description
Gets a value that indicates whether the element can be resized.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ITransformProvider.CanResize](/windows/winui/api/microsoft.ui.xaml.automation.provider.itransformprovider.canresize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the element can be resized; otherwise, **false**.
