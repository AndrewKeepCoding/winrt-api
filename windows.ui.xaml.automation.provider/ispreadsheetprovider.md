# Windows.UI.Xaml.Automation.Provider.ISpreadsheetProvider

## -description
Provides access to items (cells) in a spreadsheet.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ISpreadsheetProvider](/windows/winui/api/microsoft.ui.xaml.automation.provider.ispreadsheetprovider) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
