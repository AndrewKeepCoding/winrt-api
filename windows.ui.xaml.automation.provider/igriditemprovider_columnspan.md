# Windows.UI.Xaml.Automation.Provider.IGridItemProvider.ColumnSpan

## -description
Gets the number of columns that are spanned by a cell or item.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.IGridItemProvider.ColumnSpan](/windows/winui/api/microsoft.ui.xaml.automation.provider.igriditemprovider.columnspan) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The number of columns.
