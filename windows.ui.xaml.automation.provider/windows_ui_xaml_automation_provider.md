# Windows.UI.Xaml.Automation.Provider

## -description
Defines provider patterns that are implemented by automation peers and recognized by automation clients.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider](/windows/winui/api/microsoft.ui.xaml.automation.provider) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
