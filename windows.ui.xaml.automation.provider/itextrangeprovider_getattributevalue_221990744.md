# Windows.UI.Xaml.Automation.Provider.ITextRangeProvider.GetAttributeValue

## -description
Retrieves the value of the specified attribute ID across the text range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ITextRangeProvider.GetAttributeValue](/windows/winui/api/microsoft.ui.xaml.automation.provider.itextrangeprovider.getattributevalue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param attributeId
The text attribute ID.

## -returns
Retrieves an object that represents the value of the specified attribute.
