# Windows.UI.Xaml.Automation.Provider.ITransformProvider2.CanZoom

## -description
Gets a value that indicates whether the control supports zooming of its viewport.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ITransformProvider2.CanZoom](/windows/winui/api/microsoft.ui.xaml.automation.provider.itransformprovider2.canzoom) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the viewport can be zoomed; otherwise, **false**.
