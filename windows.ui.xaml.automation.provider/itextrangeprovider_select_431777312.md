# Windows.UI.Xaml.Automation.Provider.ITextRangeProvider.Select

## -description
Highlights text in the text control that corresponds to the start and end endpoints of the text range.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.ITextRangeProvider.Select](/windows/winui/api/microsoft.ui.xaml.automation.provider.itextrangeprovider.select) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
