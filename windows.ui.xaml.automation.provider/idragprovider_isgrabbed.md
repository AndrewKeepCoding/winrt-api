# Windows.UI.Xaml.Automation.Provider.IDragProvider.IsGrabbed

## -description
Gets a value indicating whether an item is currently being dragged.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.Provider.IDragProvider.IsGrabbed](/windows/winui/api/microsoft.ui.xaml.automation.provider.idragprovider.isgrabbed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the item is being dragged. Otherwise, **false**.
