# Windows.UI.Xaml.Controls.Control.IsFocusEngagedProperty

## -description
Identifies the [IsFocusEngaged](control_isfocusengaged.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.IsFocusEngagedProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.isfocusengagedproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsFocusEngaged](control_isfocusengaged.md) dependency property.
