# Windows.UI.Xaml.Controls.ComboBox.IsDropDownOpenProperty

## -description
Identifies the [IsDropDownOpen](combobox_isdropdownopen.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ComboBox.IsDropDownOpenProperty](/windows/winui/api/microsoft.ui.xaml.controls.combobox.isdropdownopenproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsDropDownOpen](combobox_isdropdownopen.md) dependency property.
