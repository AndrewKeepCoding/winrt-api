# Windows.UI.Xaml.Controls.MenuFlyoutSubItem.Text

## -description
Gets or sets the text content of a [MenuFlyoutSubItem](menuflyoutsubitem.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.MenuFlyoutSubItem.Text](/windows/winui/api/microsoft.ui.xaml.controls.menuflyoutsubitem.text) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
