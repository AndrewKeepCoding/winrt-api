# Windows.UI.Xaml.Controls.DatePicker.YearFormatProperty

## -description
Gets the identifier for the [YearFormat](datepicker_yearformat.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePicker.YearFormatProperty](/windows/winui/api/microsoft.ui.xaml.controls.datepicker.yearformatproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [YearFormat](datepicker_yearformat.md) dependency property.
