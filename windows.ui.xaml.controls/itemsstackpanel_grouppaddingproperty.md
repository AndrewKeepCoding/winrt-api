# Windows.UI.Xaml.Controls.ItemsStackPanel.GroupPaddingProperty

## -description
Identifies the [GroupPadding](itemsstackpanel_grouppadding.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ItemsStackPanel.GroupPaddingProperty](/windows/winui/api/microsoft.ui.xaml.controls.itemsstackpanel.grouppaddingproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [GroupPadding](itemsstackpanel_grouppadding.md) dependency property.
