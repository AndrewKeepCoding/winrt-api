# Windows.UI.Xaml.Controls.UIElementCollection.Clear

## -description
Removes all items from the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.UIElementCollection.Clear](/windows/winui/api/microsoft.ui.xaml.controls.uielementcollection.clear) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
