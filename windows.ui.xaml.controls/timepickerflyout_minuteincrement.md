# Windows.UI.Xaml.Controls.TimePickerFlyout.MinuteIncrement

## -description
Gets or sets a value that indicates the time increments shown in the minute picker. For example, 15 specifies that minute picker displays only the choices 00, 15, 30, 45.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TimePickerFlyout.MinuteIncrement](/windows/winui/api/microsoft.ui.xaml.controls.timepickerflyout.minuteincrement) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
An integer from 0-59 that indicates the increments shown in the minute picker. The default is 1.
