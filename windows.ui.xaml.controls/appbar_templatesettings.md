# Windows.UI.Xaml.Controls.AppBar.TemplateSettings

## -description
Gets an object that provides calculated values that can be referenced as [{TemplateBinding} markup extension](/windows/uwp/xaml-platform/templatebinding-markup-extension) sources when defining templates for an [AppBar](appbar.md) control.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.TemplateSettings](/windows/winui/api/microsoft.ui.xaml.controls.appbar.templatesettings) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
An object that provides calculated values for templates.
