# Windows.UI.Xaml.Controls.Page.FrameProperty

## -description
Identifies the [Frame](page_frame.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Page.FrameProperty](/windows/winui/api/microsoft.ui.xaml.controls.page.frameproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Frame](page_frame.md) dependency property.
