# Windows.UI.Xaml.Controls.ListViewItem.TemplateSettings

## -description
Gets an object that provides calculated values that can be referenced as **TemplateBinding** sources when you're defining templates for a [ListViewItem](listviewitem.md) class.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ListViewItem.TemplateSettings](/windows/winui/api/microsoft.ui.xaml.controls.listviewitem.templatesettings) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
An object that provides calculated values for templates.
