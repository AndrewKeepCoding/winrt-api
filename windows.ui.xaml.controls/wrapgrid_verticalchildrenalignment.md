# Windows.UI.Xaml.Controls.WrapGrid.VerticalChildrenAlignment

## -description
Gets or sets the alignment rules by which child elements are arranged for the vertical dimension.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.WrapGrid.VerticalChildrenAlignment](/windows/winui/api/microsoft.ui.xaml.controls.wrapgrid.verticalchildrenalignment) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<WrapGrid VerticalChildrenAlignment="verticalAlignmentMemberName" />
```


## -property-value
A value of the enumeration. The default is **Top**.
