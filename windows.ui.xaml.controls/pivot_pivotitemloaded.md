# Windows.UI.Xaml.Controls.Pivot.PivotItemLoaded

## -description
Event for indicating that an item has fully loaded.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Pivot.PivotItemLoaded](/windows/winui/api/microsoft.ui.xaml.controls.pivot.pivotitemloaded) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<Pivot PivotItemLoaded="eventhandler" />
```
