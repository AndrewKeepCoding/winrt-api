# Windows.UI.Xaml.Controls.ContentDialog.IsPrimaryButtonEnabled

## -description
Gets or sets whether the dialog's primary button is enabled.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.IsPrimaryButtonEnabled](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.isprimarybuttonenabled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml

```


## -property-value
True if the primary button of the dialog is enabled; Otherwise, false.
