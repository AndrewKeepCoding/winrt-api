# Windows.UI.Xaml.Controls.ScrollViewerView.HorizontalOffset

## -description
Gets the distance the content has been scrolled horizontally.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewerView.HorizontalOffset](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewerview.horizontaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The distance the content has been scrolled horizontally.
