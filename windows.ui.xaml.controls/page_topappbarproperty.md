# Windows.UI.Xaml.Controls.Page.TopAppBarProperty

## -description
Identifies the [TopAppBar](page_topappbar.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Page.TopAppBarProperty](/windows/winui/api/microsoft.ui.xaml.controls.page.topappbarproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [TopAppBar](page_topappbar.md) dependency property.
