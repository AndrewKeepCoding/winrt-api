# Windows.UI.Xaml.Controls.ContentDialogClosingEventArgs.GetDeferral

## -description
Gets a [ContentDialogClosingDeferral](contentdialogclosingdeferral.md) that the app can use to respond asynchronously to the closing event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialogClosingEventArgs.GetDeferral](/windows/winui/api/microsoft.ui.xaml.controls.contentdialogclosingeventargs.getdeferral) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
A [ContentDialogClosingDeferral](contentdialogclosingdeferral.md) that the app can use to respond asynchronously to the closing event.
