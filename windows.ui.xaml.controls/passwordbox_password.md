# Windows.UI.Xaml.Controls.PasswordBox.Password

## -description
Gets or sets the password currently held by the [PasswordBox](passwordbox.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.Password](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.password) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<PasswordBoxÂ Password="string"/>
```


## -property-value
A string that represents the password currently held by the [PasswordBox](passwordbox.md). The default is an empty string.
