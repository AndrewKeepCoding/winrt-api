# Windows.UI.Xaml.Controls.ContentDialog.IsSecondaryButtonEnabledProperty

## -description
Gets the identifier of the [IsSecondaryButtonEnabled](contentdialog_issecondarybuttonenabled.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.IsSecondaryButtonEnabledProperty](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.issecondarybuttonenabledproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier of the [IsSecondaryButtonEnabled](contentdialog_issecondarybuttonenabled.md) dependency property.
