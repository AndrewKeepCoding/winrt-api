# Windows.UI.Xaml.Controls.CleanUpVirtualizedItemEventArgs.Cancel

## -description
Gets or sets a value that indicates whether this item should not be revirtualized.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CleanUpVirtualizedItemEventArgs.Cancel](/windows/winui/api/microsoft.ui.xaml.controls.cleanupvirtualizeditemeventargs.cancel) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if you want to prevent revirtualization of this item; otherwise, **false**.
