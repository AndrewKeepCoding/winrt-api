# Windows.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelRight

## -description
Scrolls right within content after a user clicks the wheel button on a mouse.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelRight](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.mousewheelright) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
