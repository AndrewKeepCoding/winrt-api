# Windows.UI.Xaml.Controls.CalendarDatePicker.Date

## -description
Gets or sets the date currently set in the calendar picker.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CalendarDatePicker.Date](/windows/winui/api/microsoft.ui.xaml.controls.calendardatepicker.date) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The date currently set in the calendar picker.
