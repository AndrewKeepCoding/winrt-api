# Windows.UI.Xaml.Controls.ContentPresenter.HorizontalContentAlignmentProperty

## -description
Identifies the [HorizontalContentAlignment](contentpresenter_horizontalcontentalignment.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentPresenter.HorizontalContentAlignmentProperty](/windows/winui/api/microsoft.ui.xaml.controls.contentpresenter.horizontalcontentalignmentproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [HorizontalContentAlignment](contentpresenter_horizontalcontentalignment.md) dependency property.
