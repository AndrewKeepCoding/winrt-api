# Windows.UI.Xaml.Controls.TextBlock.TextTrimmingProperty

## -description
Identifies the [TextTrimming](textblock_texttrimming.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBlock.TextTrimmingProperty](/windows/winui/api/microsoft.ui.xaml.controls.textblock.texttrimmingproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier of the [TextTrimming](textblock_texttrimming.md) dependency property.
