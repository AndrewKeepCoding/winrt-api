# Windows.UI.Xaml.Controls.Panel.BackgroundProperty

## -description
Identifies the [Background](panel_background.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Panel.BackgroundProperty](/windows/winui/api/microsoft.ui.xaml.controls.panel.backgroundproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Background](panel_background.md)  dependency property.
