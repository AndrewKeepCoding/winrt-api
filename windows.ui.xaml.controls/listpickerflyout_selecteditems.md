# Windows.UI.Xaml.Controls.ListPickerFlyout.SelectedItems

## -description
Gets the list of currently selected items.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ListPickerFlyout.SelectedItems](/windows/winui/api/microsoft.ui.xaml.controls.listpickerflyout.selecteditems) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The list of currently selected items.
