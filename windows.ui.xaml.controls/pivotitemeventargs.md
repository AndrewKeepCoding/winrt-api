# Windows.UI.Xaml.Controls.PivotItemEventArgs

## -description
Event arguments for dynamically interacting with the [PivotItem](pivotitem.md) before use, allowing for delay load scenarios.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PivotItemEventArgs](/windows/winui/api/microsoft.ui.xaml.controls.pivotitemeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
