# Windows.UI.Xaml.Controls.VirtualizingPanel.BringIndexIntoView

## -description
Generates the item at the specified index location and makes it visible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.VirtualizingPanel.BringIndexIntoView](/windows/winui/api/microsoft.ui.xaml.controls.virtualizingpanel.bringindexintoview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index position of the item that is generated and made visible.
