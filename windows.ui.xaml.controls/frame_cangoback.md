# Windows.UI.Xaml.Controls.Frame.CanGoBack

## -description
Gets a value that indicates whether there is at least one entry in back navigation history.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.CanGoBack](/windows/winui/api/microsoft.ui.xaml.controls.frame.cangoback) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if there is at least one entry in back navigation history; **false** if there are no entries in back navigation history or the [Frame](frame.md) does not own its own navigation history.
