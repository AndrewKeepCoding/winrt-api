# Windows.UI.Xaml.Controls.AppBar.IsStickyProperty

## -description
Identifies the [IsSticky](appbar_issticky.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.IsStickyProperty](/windows/winui/api/microsoft.ui.xaml.controls.appbar.isstickyproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsSticky](appbar_issticky.md) dependency property.
