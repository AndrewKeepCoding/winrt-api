# Windows.UI.Xaml.Controls.TextBoxSelectionChangingEventArgs.SelectionLength

## -description

Gets the length of the text selection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBoxSelectionChangingEventArgs.SelectionLength](/windows/winui/api/microsoft.ui.xaml.controls.textboxselectionchangingeventargs.selectionlength) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The length of the text selection.
