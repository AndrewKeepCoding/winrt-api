# Windows.UI.Xaml.Controls.RichEditBoxSelectionChangingEventArgs.SelectionStart

## -description

Gets the starting index of the text selection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichEditBoxSelectionChangingEventArgs.SelectionStart](/windows/winui/api/microsoft.ui.xaml.controls.richeditboxselectionchangingeventargs.selectionstart) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The starting index of the text selection.
