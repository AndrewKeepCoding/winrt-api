# Windows.UI.Xaml.Controls.RichTextBlock.SelectedTextProperty

## -description
Identifies the [SelectedText](richtextblock_selectedtext.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlock.SelectedTextProperty](/windows/winui/api/microsoft.ui.xaml.controls.richtextblock.selectedtextproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [SelectedText](richtextblock_selectedtext.md) dependency property.
