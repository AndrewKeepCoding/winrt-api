# Windows.UI.Xaml.Controls.StackPanel.SpacingProperty

## -description

Identifies the Spacing dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.StackPanel.SpacingProperty](/windows/winui/api/microsoft.ui.xaml.controls.stackpanel.spacingproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the Spacing dependency property.
