# Windows.UI.Xaml.Controls.Frame.Navigating

## -description
Occurs when a new navigation is requested.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.Navigating](/windows/winui/api/microsoft.ui.xaml.controls.frame.navigating) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
