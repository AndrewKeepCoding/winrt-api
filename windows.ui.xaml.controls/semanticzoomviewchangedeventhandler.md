# Windows.UI.Xaml.Controls.SemanticZoomViewChangedEventHandler

## -description
Represents the method that will handle the [ViewChangeStarted](semanticzoom_viewchangestarted.md) and [ViewChangeCompleted](semanticzoom_viewchangecompleted.md) events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.SemanticZoomViewChangedEventHandler](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoomviewchangedeventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The object where the handler is attached.

### -param e
Event data for the event.
