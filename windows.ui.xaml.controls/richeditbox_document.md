# Windows.UI.Xaml.Controls.RichEditBox.Document

## -description
Gets an object that enables access to the text object model for the text contained in a [RichEditBox](richeditbox.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichEditBox.Document](/windows/winui/api/microsoft.ui.xaml.controls.richeditbox.document) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
An object that enables access to the text object model.
