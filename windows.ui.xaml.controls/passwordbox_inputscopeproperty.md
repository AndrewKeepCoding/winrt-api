# Windows.UI.Xaml.Controls.PasswordBox.InputScopeProperty

## -description
Identifies the [InputScope](passwordbox_inputscope.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.InputScopeProperty](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.inputscopeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [InputScope](passwordbox_inputscope.md) dependency property.
