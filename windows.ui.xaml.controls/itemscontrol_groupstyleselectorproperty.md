# Windows.UI.Xaml.Controls.ItemsControl.GroupStyleSelectorProperty

## -description
Identifies the [GroupStyleSelector](itemscontrol_groupstyleselector.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ItemsControl.GroupStyleSelectorProperty](/windows/winui/api/microsoft.ui.xaml.controls.itemscontrol.groupstyleselectorproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [GroupStyleSelector](itemscontrol_groupstyleselector.md) dependency property.
