# Windows.UI.Xaml.Controls.Border.BorderThicknessProperty

## -description
Identifies the [BorderThickness](border_borderthickness.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Border.BorderThicknessProperty](/windows/winui/api/microsoft.ui.xaml.controls.border.borderthicknessproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [BorderThickness](border_borderthickness.md) dependency property.
