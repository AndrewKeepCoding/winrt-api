# Windows.UI.Xaml.Controls.ItemsPresenter.FooterTemplateProperty

## -description
Identifies the [FooterTemplate](itemspresenter_footertemplate.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ItemsPresenter.FooterTemplateProperty](/windows/winui/api/microsoft.ui.xaml.controls.itemspresenter.footertemplateproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [FooterTemplate](itemspresenter_footertemplate.md) dependency property.
