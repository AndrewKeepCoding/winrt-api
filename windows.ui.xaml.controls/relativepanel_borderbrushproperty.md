# Windows.UI.Xaml.Controls.RelativePanel.BorderBrushProperty

## -description
Identifies the [BorderBrush](relativepanel_borderbrush.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RelativePanel.BorderBrushProperty](/windows/winui/api/microsoft.ui.xaml.controls.relativepanel.borderbrushproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [BorderBrush](relativepanel_borderbrush.md) dependency property.
