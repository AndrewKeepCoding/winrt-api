# Windows.UI.Xaml.Controls.AutoSuggestBoxQuerySubmittedEventArgs.QueryText

## -description
Gets the query text of the current search.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AutoSuggestBoxQuerySubmittedEventArgs.QueryText](/windows/winui/api/microsoft.ui.xaml.controls.autosuggestboxquerysubmittedeventargs.querytext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The query text of the current search.
