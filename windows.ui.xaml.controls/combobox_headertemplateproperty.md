# Windows.UI.Xaml.Controls.ComboBox.HeaderTemplateProperty

## -description
Identifies the [HeaderTemplate](combobox_headertemplate.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ComboBox.HeaderTemplateProperty](/windows/winui/api/microsoft.ui.xaml.controls.combobox.headertemplateproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [HeaderTemplate](combobox_headertemplate.md) dependency property.
