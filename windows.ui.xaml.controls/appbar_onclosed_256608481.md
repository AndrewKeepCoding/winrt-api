# Windows.UI.Xaml.Controls.AppBar.OnClosed

## -description
Invoked when the [AppBar](appbar.md) changes from visible to hidden.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.OnClosed](/windows/winui/api/microsoft.ui.xaml.controls.appbar.onclosed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param e
Event data for the event.
