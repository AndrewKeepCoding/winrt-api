# Windows.UI.Xaml.Controls.Pivot.RightHeaderTemplateProperty

## -description
Identifies the [RightHeaderTemplate](pivot_rightheadertemplate.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Pivot.RightHeaderTemplateProperty](/windows/winui/api/microsoft.ui.xaml.controls.pivot.rightheadertemplateproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [RightHeaderTemplate](pivot_rightheadertemplate.md) dependency property.
