# Windows.UI.Xaml.Controls.ISemanticZoomInformation.CompleteViewChangeTo

## -description
Completes item-wise operations related to a view change when the implementing view is the destination view and the source view is a potentially different implementing view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.CompleteViewChangeTo](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.completeviewchangeto) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param source
The view item as represented in the source view.

### -param destination
The view item as represented in the destination view.
