# Windows.UI.Xaml.Controls.CommandBar.PrimaryCommandsProperty

## -description
Identifies the [PrimaryCommands](commandbar_primarycommands.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CommandBar.PrimaryCommandsProperty](/windows/winui/api/microsoft.ui.xaml.controls.commandbar.primarycommandsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [PrimaryCommands](commandbar_primarycommands.md) dependency property.
