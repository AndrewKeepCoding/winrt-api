# Windows.UI.Xaml.Controls.StackPanel.BorderThicknessProperty

## -description
Identifies the [BorderThickness](stackpanel_borderthickness.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.StackPanel.BorderThicknessProperty](/windows/winui/api/microsoft.ui.xaml.controls.stackpanel.borderthicknessproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [BorderThickness](stackpanel_borderthickness.md) dependency property.
