# Windows.UI.Xaml.Controls.TextBox.CopySelectionToClipboard

## -description

Copies the selected content to the Windows clipboard.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.CopySelectionToClipboard](/windows/winui/api/microsoft.ui.xaml.controls.textbox.copyselectiontoclipboard) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
