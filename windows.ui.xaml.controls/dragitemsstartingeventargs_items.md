# Windows.UI.Xaml.Controls.DragItemsStartingEventArgs.Items

## -description
Gets the loosely typed collection of objects that are selected for the item drag action.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DragItemsStartingEventArgs.Items](/windows/winui/api/microsoft.ui.xaml.controls.dragitemsstartingeventargs.items).

## -property-value
A loosely typed collection of objects.
