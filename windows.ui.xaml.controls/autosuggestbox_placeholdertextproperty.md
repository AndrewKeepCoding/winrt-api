# Windows.UI.Xaml.Controls.AutoSuggestBox.PlaceholderTextProperty

## -description
Identifies the [PlaceholderText](autosuggestbox_placeholdertext.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AutoSuggestBox.PlaceholderTextProperty](/windows/winui/api/microsoft.ui.xaml.controls.autosuggestbox.placeholdertextproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [PlaceholderText](autosuggestbox_placeholdertext.md) dependency property.
