# Windows.UI.Xaml.Controls.RowDefinitionCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RowDefinitionCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.controls.rowdefinitioncollection.replaceall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param items
The new collection items.
