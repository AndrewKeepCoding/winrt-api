# Windows.UI.Xaml.Controls.TextBlock.ContextMenuOpening

## -description
Occurs when the system processes an interaction that displays a context menu.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBlock.ContextMenuOpening](/windows/winui/api/microsoft.ui.xaml.controls.textblock.contextmenuopening) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<TextBlock ContextMenuOpening="eventhandler" />
```
