# Windows.UI.Xaml.Controls.ScrollViewer.HorizontalOffsetProperty

## -description
Identifies the [HorizontalOffset](scrollviewer_horizontaloffset.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.HorizontalOffsetProperty](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.horizontaloffsetproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [HorizontalOffset](scrollviewer_horizontaloffset.md)  dependency property.
