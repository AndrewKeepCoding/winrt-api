# Windows.UI.Xaml.Controls.ScrollViewer.LeftHeaderProperty

## -description
Identifies the [LeftHeader](scrollviewer_leftheader.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.LeftHeaderProperty](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.leftheaderproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [LeftHeader](scrollviewer_leftheader.md) dependency property.
