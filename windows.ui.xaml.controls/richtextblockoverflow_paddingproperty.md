# Windows.UI.Xaml.Controls.RichTextBlockOverflow.PaddingProperty

## -description
Identifies the [Padding](richtextblockoverflow_padding.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlockOverflow.PaddingProperty](/windows/winui/api/microsoft.ui.xaml.controls.richtextblockoverflow.paddingproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Padding](richtextblockoverflow_padding.md) dependency property.
