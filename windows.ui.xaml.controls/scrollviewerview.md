# Windows.UI.Xaml.Controls.ScrollViewerView

## -description
Provides a view for a [ScrollViewer](scrollviewer.md) when its view is changing.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewerView](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewerview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
