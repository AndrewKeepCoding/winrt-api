# Windows.UI.Xaml.Controls.AppBar.ClosedDisplayModeProperty

## -description
Identifies the [ClosedDisplayMode](appbar_closeddisplaymode.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.ClosedDisplayModeProperty](/windows/winui/api/microsoft.ui.xaml.controls.appbar.closeddisplaymodeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ClosedDisplayMode](appbar_closeddisplaymode.md) dependency property.
