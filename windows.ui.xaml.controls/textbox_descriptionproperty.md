# Windows.UI.Xaml.Controls.TextBox.DescriptionProperty

## -description

Identifies the Description dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.DescriptionProperty](/windows/winui/api/microsoft.ui.xaml.controls.textbox.descriptionproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the Description dependency property.
