# Windows.UI.Xaml.Controls.TimePickerSelectedValueChangedEventArgs.OldTime

## -description

Gets the time previously selected in the picker.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TimePickerSelectedValueChangedEventArgs.OldTime](/windows/winui/api/microsoft.ui.xaml.controls.timepickerselectedvaluechangedeventargs.oldtime) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The time previously selected in the picker.
