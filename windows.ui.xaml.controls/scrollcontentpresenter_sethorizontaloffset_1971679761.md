# Windows.UI.Xaml.Controls.ScrollContentPresenter.SetHorizontalOffset

## -description
Sets the distance the content has been scrolled horizontally.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.SetHorizontalOffset](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.sethorizontaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param offset
The distance the content has been scrolled horizontally.
