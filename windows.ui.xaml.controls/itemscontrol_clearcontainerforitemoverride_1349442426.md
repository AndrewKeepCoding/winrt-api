# Windows.UI.Xaml.Controls.ItemsControl.ClearContainerForItemOverride

## -description
Undoes the effects of the [PrepareContainerForItemOverride](itemscontrol_preparecontainerforitemoverride_1839050347.md) method.

### -param element
