# Windows.UI.Xaml.Controls.RichEditBox.GetLinguisticAlternativesAsync

## -description
Asynchronously gets a list of candidate words based on the provided phonetic characters in an Input Method Editor (IME).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichEditBox.GetLinguisticAlternativesAsync](/windows/winui/api/microsoft.ui.xaml.controls.richeditbox.getlinguisticalternativesasync) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
A list of candidate words based on the provided phonetic characters in an Input Method Editor (IME).
