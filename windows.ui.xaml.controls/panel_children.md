# Windows.UI.Xaml.Controls.Panel.Children

## -description
Gets the collection of child elements of the panel.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Panel.Children](/windows/winui/api/microsoft.ui.xaml.controls.panel.children) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<panel>
  oneOrMoreUIElements
</panel>
```


## -xaml-values
<dl><dt>oneOrMoreUIElements</dt><dd>oneOrMoreUIElementsOne or more object elements for objects that derive from UIElement.</dd>
</dl>
## -property-value
The collection of child objects. The default is an empty collection.
