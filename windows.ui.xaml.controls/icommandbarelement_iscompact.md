# Windows.UI.Xaml.Controls.ICommandBarElement.IsCompact

## -description
Gets or sets a value that indicates whether the element is shown with no label and reduced padding.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ICommandBarElement.IsCompact](/windows/winui/api/microsoft.ui.xaml.controls.icommandbarelement.iscompact) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the element is shown in its compact state; otherwise, **false**. The default is **false**.
