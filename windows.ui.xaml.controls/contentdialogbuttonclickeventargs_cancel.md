# Windows.UI.Xaml.Controls.ContentDialogButtonClickEventArgs.Cancel

## -description
Gets or sets a value that can cancel the button click. A **true** value for Cancel cancels the default behavior.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialogButtonClickEventArgs.Cancel](/windows/winui/api/microsoft.ui.xaml.controls.contentdialogbuttonclickeventargs.cancel) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
True to cancel the button click; Otherwise, false.
