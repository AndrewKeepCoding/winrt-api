# Windows.UI.Xaml.Controls.AppBarButton.IconProperty

## -description
Identifies the [Icon](appbarbutton_icon.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBarButton.IconProperty](/windows/winui/api/microsoft.ui.xaml.controls.appbarbutton.iconproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Icon](appbarbutton_icon.md) dependency property.
