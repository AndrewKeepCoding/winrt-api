# Windows.UI.Xaml.Controls.ColumnDefinition.WidthProperty

## -description
Identifies the [Width](columndefinition_width.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ColumnDefinition.WidthProperty](/windows/winui/api/microsoft.ui.xaml.controls.columndefinition.widthproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Width](columndefinition_width.md) dependency property.
