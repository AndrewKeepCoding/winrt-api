# Windows.UI.Xaml.Controls.DatePickerFlyout.Date

## -description
Gets or sets the date currently set in the date picker.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePickerFlyout.Date](/windows/winui/api/microsoft.ui.xaml.controls.datepickerflyout.date) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The date currently set in the picker.
