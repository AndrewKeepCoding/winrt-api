# Windows.UI.Xaml.Controls.ToggleSwitch.IsOnProperty

## -description
Identifies the [IsOn](toggleswitch_ison.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ToggleSwitch.IsOnProperty](/windows/winui/api/microsoft.ui.xaml.controls.toggleswitch.isonproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsOn](toggleswitch_ison.md) dependency property.
