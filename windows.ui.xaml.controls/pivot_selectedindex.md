# Windows.UI.Xaml.Controls.Pivot.SelectedIndex

## -description
Gets or sets the zero-based index of the currently selected item in the [Pivot](pivot.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Pivot.SelectedIndex](/windows/winui/api/microsoft.ui.xaml.controls.pivot.selectedindex) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The zero-based index of the currently selected item.
