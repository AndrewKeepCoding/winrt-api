# Windows.UI.Xaml.Controls.ComboBox.IsDropDownOpen

## -description
Gets or sets a value that indicates whether the drop-down portion of the [ComboBox](combobox.md) is currently open.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ComboBox.IsDropDownOpen](/windows/winui/api/microsoft.ui.xaml.controls.combobox.isdropdownopen) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<ComboBoxÂ IsDropDownOpen="bool"/>
```


## -property-value
**True** if the drop-down portion is open; otherwise, **false**. The default is **false**.
