# Windows.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelLeft

## -description
Scrolls left within content after a user clicks the wheel button on a mouse.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelLeft](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.mousewheelleft) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
