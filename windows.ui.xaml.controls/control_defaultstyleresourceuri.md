# Windows.UI.Xaml.Controls.Control.DefaultStyleResourceUri

## -description
Gets or sets the path to the resource file that contains the default style for the control.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.DefaultStyleResourceUri](/windows/winui/api/microsoft.ui.xaml.controls.control.defaultstyleresourceuri) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The path to the resource file that contains the default style for the control.
