# Windows.UI.Xaml.Controls.Control.OnDragEnter

## -description
Called before the [DragEnter](../windows.ui.xaml/uielement_dragenter.md) event occurs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.OnDragEnter](/windows/winui/api/microsoft.ui.xaml.controls.control.ondragenter) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param e
Event data for the event.
