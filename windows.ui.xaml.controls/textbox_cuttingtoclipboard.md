# Windows.UI.Xaml.Controls.TextBox.CuttingToClipboard

## -description

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.CuttingToClipboard](/windows/winui/api/microsoft.ui.xaml.controls.textbox.cuttingtoclipboard) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
