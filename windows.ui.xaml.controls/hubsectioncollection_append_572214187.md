# Windows.UI.Xaml.Controls.HubSectionCollection.Append

## -description
Adds a new item to the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.HubSectionCollection.Append](/windows/winui/api/microsoft.ui.xaml.controls.hubsectioncollection.append) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param value
The new item to add.

## -remarks
To add an item to a position in the collection that's not at the collection end, use [InsertAt](hubsectioncollection_insertat_791846136.md).
