# Windows.UI.Xaml.Controls.TimePickerFlyout.ClockIdentifier

## -description
Gets or sets the clock system to use.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TimePickerFlyout.ClockIdentifier](/windows/winui/api/microsoft.ui.xaml.controls.timepickerflyout.clockidentifier) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The name of the clock system to use.
