# Windows.UI.Xaml.Controls.DatePicker.MonthVisibleProperty

## -description
Gets the identifier for the [MonthVisible](datepicker_monthvisible.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePicker.MonthVisibleProperty](/windows/winui/api/microsoft.ui.xaml.controls.datepicker.monthvisibleproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MonthVisible](datepicker_monthvisible.md) dependency property.
