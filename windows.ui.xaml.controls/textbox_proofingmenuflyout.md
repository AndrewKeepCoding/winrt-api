# Windows.UI.Xaml.Controls.TextBox.ProofingMenuFlyout

## -description

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.ProofingMenuFlyout](/windows/winui/api/microsoft.ui.xaml.controls.textbox.proofingmenuflyout) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The flyout that shows proofing commands.
