# Windows.UI.Xaml.Controls.ListViewBase.IsMultiSelectCheckBoxEnabled

## -description
Gets or sets a value that indicates whether a check box is shown to enable multi-selection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ListViewBase.IsMultiSelectCheckBoxEnabled](/windows/winui/api/microsoft.ui.xaml.controls.listviewbase.ismultiselectcheckboxenabled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if a check box is shown to enable multi-selection; otherwise, **false**.
