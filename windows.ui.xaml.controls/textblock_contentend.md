# Windows.UI.Xaml.Controls.TextBlock.ContentEnd

## -description
Gets a [TextPointer](../windows.ui.xaml.documents/textpointer.md) object for the end of text content in the [TextBlock](textblock.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBlock.ContentEnd](/windows/winui/api/microsoft.ui.xaml.controls.textblock.contentend) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A [TextPointer](../windows.ui.xaml.documents/textpointer.md) object for the end of text content in the [TextBlock](textblock.md).
