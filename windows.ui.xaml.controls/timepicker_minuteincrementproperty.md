# Windows.UI.Xaml.Controls.TimePicker.MinuteIncrementProperty

## -description
Gets the identifier for the [MinuteIncrement](timepicker_minuteincrement.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TimePicker.MinuteIncrementProperty](/windows/winui/api/microsoft.ui.xaml.controls.timepicker.minuteincrementproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MinuteIncrement](timepicker_minuteincrement.md) dependency property.
