# Windows.UI.Xaml.Controls.ScrollViewer.ScrollableHeightProperty

## -description
Identifies the [ScrollableHeight](scrollviewer_scrollableheight.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.ScrollableHeightProperty](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.scrollableheightproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ScrollableHeight](scrollviewer_scrollableheight.md)  dependency property.
