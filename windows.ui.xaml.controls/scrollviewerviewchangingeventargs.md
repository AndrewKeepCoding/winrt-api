# Windows.UI.Xaml.Controls.ScrollViewerViewChangingEventArgs

## -description
Provides event data for the [ViewChanging](scrollviewer_viewchanging.md) event and similar control-specific events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewerViewChangingEventArgs](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewerviewchangingeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
