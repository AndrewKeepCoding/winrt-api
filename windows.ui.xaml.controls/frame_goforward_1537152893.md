# Windows.UI.Xaml.Controls.Frame.GoForward

## -description
Navigates to the most recent item in forward navigation history, if a [Frame](frame.md) manages its own navigation history.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.GoForward](/windows/winui/api/microsoft.ui.xaml.controls.frame.goforward) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
