# Windows.UI.Xaml.Controls.Frame.GoBack

## -description
Navigates to the most recent item in back navigation history, if a [Frame](frame.md) manages its own navigation history, and specifies the animated transition to use.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.GoBack](/windows/winui/api/microsoft.ui.xaml.controls.frame.goback) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param transitionInfoOverride
Info about the animated transition to use.
