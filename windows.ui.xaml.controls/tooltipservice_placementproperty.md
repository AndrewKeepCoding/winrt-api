# Windows.UI.Xaml.Controls.ToolTipService.PlacementProperty

## -description
Identifies the [ToolTipService.Placement](tooltipservice_placement.md) XAML attached property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ToolTipService.PlacementProperty](/windows/winui/api/microsoft.ui.xaml.controls.tooltipservice.placementproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ToolTipService.Placement](tooltipservice_placement.md) XAML attached property.
