# Windows.UI.Xaml.Controls.FontIcon.FontStyle

## -description
Gets or sets the font style for the icon glyph.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.FontIcon.FontStyle](/windows/winui/api/microsoft.ui.xaml.controls.fonticon.fontstyle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
