# Windows.UI.Xaml.Controls.CalendarView.SetDisplayDate

## -description
Shows the specified date in the calendar.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CalendarView.SetDisplayDate](/windows/winui/api/microsoft.ui.xaml.controls.calendarview.setdisplaydate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param date
The date to show in the calendar.
