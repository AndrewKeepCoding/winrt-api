# Windows.UI.Xaml.Controls.CommandBar.SecondaryCommandsProperty

## -description
Identifies the [SecondaryCommands](commandbar_secondarycommands.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CommandBar.SecondaryCommandsProperty](/windows/winui/api/microsoft.ui.xaml.controls.commandbar.secondarycommandsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [SecondaryCommands](commandbar_secondarycommands.md) dependency property.
