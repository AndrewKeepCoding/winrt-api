# Windows.UI.Xaml.Controls.ContentDialogButtonClickDeferral.Complete

## -description
Notifies the system that the app has finished processing the button click event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialogButtonClickDeferral.Complete](/windows/winui/api/microsoft.ui.xaml.controls.contentdialogbuttonclickdeferral.complete) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
