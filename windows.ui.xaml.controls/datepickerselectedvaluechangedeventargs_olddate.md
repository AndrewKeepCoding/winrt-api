# Windows.UI.Xaml.Controls.DatePickerSelectedValueChangedEventArgs.OldDate

## -description

Gets the date previously selected in the picker.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePickerSelectedValueChangedEventArgs.OldDate](/windows/winui/api/microsoft.ui.xaml.controls.datepickerselectedvaluechangedeventargs.olddate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The date previously selected in the picker.
