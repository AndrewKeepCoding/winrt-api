# Windows.UI.Xaml.Controls.FontIcon.GlyphProperty

## -description
Gets the identifier for the [Glyph](fonticon_glyph.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.FontIcon.GlyphProperty](/windows/winui/api/microsoft.ui.xaml.controls.fonticon.glyphproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Glyph](fonticon_glyph.md) dependency property.
