# Windows.UI.Xaml.Controls.ComboBox.SelectionBoxItemTemplate

## -description
Gets the template applied to the selection box content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ComboBox.SelectionBoxItemTemplate](/windows/winui/api/microsoft.ui.xaml.controls.combobox.selectionboxitemtemplate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The template applied to the selection box content.
