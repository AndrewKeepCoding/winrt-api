# Windows.UI.Xaml.Controls.TextBox.SelectAll

## -description
Selects the entire contents of the text box.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.SelectAll](/windows/winui/api/microsoft.ui.xaml.controls.textbox.selectall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
