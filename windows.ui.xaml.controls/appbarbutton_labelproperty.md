# Windows.UI.Xaml.Controls.AppBarButton.LabelProperty

## -description
Identifies the [Label](appbarbutton_label.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBarButton.LabelProperty](/windows/winui/api/microsoft.ui.xaml.controls.appbarbutton.labelproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Label](appbarbutton_label.md) dependency property.
