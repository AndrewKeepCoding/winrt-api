# Windows.UI.Xaml.Controls.ContentPresenter.TextWrapping

## -description
Gets or sets how the element wraps text.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentPresenter.TextWrapping](/windows/winui/api/microsoft.ui.xaml.controls.contentpresenter.textwrapping) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that indicates how the element wraps text. The default is **NoWrap**.
