# Windows.UI.Xaml.Controls.ContentPresenter.FontSize

## -description
Gets or sets the font size for the text content presented by the [ContentPresenter](contentpresenter.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentPresenter.FontSize](/windows/winui/api/microsoft.ui.xaml.controls.contentpresenter.fontsize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml

```


## -property-value
A non-negative value that specifies the font size, measured in pixels.
