# Windows.UI.Xaml.Controls.AppBar.OnOpened

## -description
Invoked when the [AppBar](appbar.md) changes from hidden to visible, or is first displayed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBar.OnOpened](/windows/winui/api/microsoft.ui.xaml.controls.appbar.onopened) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param e
Event data for the event.
