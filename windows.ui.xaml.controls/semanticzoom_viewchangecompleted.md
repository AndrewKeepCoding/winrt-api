# Windows.UI.Xaml.Controls.SemanticZoom.ViewChangeCompleted

## -description
Occurs when a view change is complete and the view is displayed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.SemanticZoom.ViewChangeCompleted](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoom.viewchangecompleted) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<SemanticZoom ViewChangeCompleted="eventhandler" />
```
