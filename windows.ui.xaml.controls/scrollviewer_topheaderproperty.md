# Windows.UI.Xaml.Controls.ScrollViewer.TopHeaderProperty

## -description
Identifies the [TopHeader](scrollviewer_topheader.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.TopHeaderProperty](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.topheaderproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [TopHeader](scrollviewer_topheader.md) dependency property.
