# Windows.UI.Xaml.Controls.DatePicker.MaxYearProperty

## -description
Gets the identifier for the [MaxYear](datepicker_maxyear.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePicker.MaxYearProperty](/windows/winui/api/microsoft.ui.xaml.controls.datepicker.maxyearproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MaxYear](datepicker_maxyear.md) dependency property.
