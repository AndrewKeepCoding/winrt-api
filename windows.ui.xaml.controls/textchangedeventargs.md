# Windows.UI.Xaml.Controls.TextChangedEventArgs

## -description

Provides data for the [TextChanged](textbox_textchanged.md) event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextChangedEventArgs](/windows/winui/api/microsoft.ui.xaml.controls.textchangedeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
