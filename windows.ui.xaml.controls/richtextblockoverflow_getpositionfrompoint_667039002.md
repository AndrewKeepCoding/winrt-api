# Windows.UI.Xaml.Controls.RichTextBlockOverflow.GetPositionFromPoint

## -description
Returns a [TextPointer](../windows.ui.xaml.documents/textpointer.md) text reference from a [RichTextBlockOverflow](richtextblockoverflow.md) by hit-testing a specific [Point](../windows.foundation/point.md) within the text display area.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlockOverflow.GetPositionFromPoint](/windows/winui/api/microsoft.ui.xaml.controls.richtextblockoverflow.getpositionfrompoint) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param point
The point to test.

## -returns
The determined text reference.
