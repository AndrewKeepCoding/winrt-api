# Windows.UI.Xaml.Controls.Panel.IsItemsHostProperty

## -description
Identifies the [IsItemsHost](panel_isitemshost.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Panel.IsItemsHostProperty](/windows/winui/api/microsoft.ui.xaml.controls.panel.isitemshostproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsItemsHost](panel_isitemshost.md) dependency property.
