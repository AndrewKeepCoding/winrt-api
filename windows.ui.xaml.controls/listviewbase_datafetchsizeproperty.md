# Windows.UI.Xaml.Controls.ListViewBase.DataFetchSizeProperty

## -description
Identifies the [DataFetchSize](listviewbase_datafetchsize.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ListViewBase.DataFetchSizeProperty](/windows/winui/api/microsoft.ui.xaml.controls.listviewbase.datafetchsizeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [DataFetchSize](listviewbase_datafetchsize.md) dependency property.
