# Windows.UI.Xaml.Controls.Control.BackgroundProperty

## -description
Identifies the [Background](control_background.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.BackgroundProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.backgroundproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Background](control_background.md) dependency property.
