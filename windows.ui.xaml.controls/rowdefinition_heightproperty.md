# Windows.UI.Xaml.Controls.RowDefinition.HeightProperty

## -description
Identifies the [Height](rowdefinition_height.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RowDefinition.HeightProperty](/windows/winui/api/microsoft.ui.xaml.controls.rowdefinition.heightproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Height](rowdefinition_height.md) dependency property.
