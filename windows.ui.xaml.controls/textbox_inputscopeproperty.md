# Windows.UI.Xaml.Controls.TextBox.InputScopeProperty

## -description
Identifies the InputScope dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.InputScopeProperty](/windows/winui/api/microsoft.ui.xaml.controls.textbox.inputscopeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the InputScope dependency property.
