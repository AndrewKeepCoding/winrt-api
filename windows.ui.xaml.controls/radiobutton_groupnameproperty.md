# Windows.UI.Xaml.Controls.RadioButton.GroupNameProperty

## -description
Identifies the [GroupName](radiobutton_groupname.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RadioButton.GroupNameProperty](/windows/winui/api/microsoft.ui.xaml.controls.radiobutton.groupnameproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for [GroupName](radiobutton_groupname.md) dependency property.
