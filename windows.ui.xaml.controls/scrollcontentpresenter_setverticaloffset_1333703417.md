# Windows.UI.Xaml.Controls.ScrollContentPresenter.SetVerticalOffset

## -description
Sets the distance the content has been scrolled vertically.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.SetVerticalOffset](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.setverticaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param offset
The distance the content has been scrolled vertically.
