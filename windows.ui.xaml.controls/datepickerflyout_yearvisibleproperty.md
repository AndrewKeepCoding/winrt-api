# Windows.UI.Xaml.Controls.DatePickerFlyout.YearVisibleProperty

## -description
Gets the identifier for the [YearVisible](datepickerflyout_yearvisible.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePickerFlyout.YearVisibleProperty](/windows/winui/api/microsoft.ui.xaml.controls.datepickerflyout.yearvisibleproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [YearVisible](datepickerflyout_yearvisible.md) dependency property.
