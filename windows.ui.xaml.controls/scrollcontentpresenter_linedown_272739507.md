# Windows.UI.Xaml.Controls.ScrollContentPresenter.LineDown

## -description
Scrolls the [ScrollContentPresenter](scrollcontentpresenter.md) content downward by one line.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.LineDown](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.linedown) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
