# Windows.UI.Xaml.Controls.ContentControl.ContentTemplateRoot

## -description
Gets the root element of the data template specified by the [ContentTemplate](contentcontrol_contenttemplate.md) property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentControl.ContentTemplateRoot](/windows/winui/api/microsoft.ui.xaml.controls.contentcontrol.contenttemplateroot) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The root element of the data template specified by the [ContentTemplate](contentcontrol_contenttemplate.md) property. The default is **null**.
