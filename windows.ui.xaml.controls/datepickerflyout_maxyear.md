# Windows.UI.Xaml.Controls.DatePickerFlyout.MaxYear

## -description
Gets or sets the maximum Gregorian year available for picking.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.DatePickerFlyout.MaxYear](/windows/winui/api/microsoft.ui.xaml.controls.datepickerflyout.maxyear) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The maximum Gregorian year available for picking.S
