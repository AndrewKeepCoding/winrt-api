# Windows.UI.Xaml.Controls.Slider.HeaderProperty

## -description
Identifies the [Header](slider_header.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Slider.HeaderProperty](/windows/winui/api/microsoft.ui.xaml.controls.slider.headerproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Header](slider_header.md) dependency property.
