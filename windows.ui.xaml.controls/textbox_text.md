# Windows.UI.Xaml.Controls.TextBox.Text

## -description
Gets or sets the text contents of the text box.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.Text](/windows/winui/api/microsoft.ui.xaml.controls.textbox.text) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A string containing the text contents of the text box. The default is an empty string ("").
