# Windows.UI.Xaml.Controls.RichTextBlock.TextLineBoundsProperty

## -description
Identifies the [TextLineBounds](richtextblock_textlinebounds.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlock.TextLineBoundsProperty](/windows/winui/api/microsoft.ui.xaml.controls.richtextblock.textlineboundsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [TextLineBounds](richtextblock_textlinebounds.md) dependency property.
