# Windows.UI.Xaml.Controls.PasswordBox.PasswordCharProperty

## -description
Identifies the [PasswordChar](passwordbox_passwordchar.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.PasswordCharProperty](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.passwordcharproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [PasswordChar](passwordbox_passwordchar.md) dependency property.
