# Windows.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs.IsSourceZoomedInView

## -description
Gets or sets a value that indicates whether the starting view is the [ZoomedInView](semanticzoom_zoomedinview.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs.IsSourceZoomedInView](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoomviewchangedeventargs.issourcezoomedinview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the starting view is the [ZoomedInView](semanticzoom_zoomedinview.md); otherwise, **false**.
