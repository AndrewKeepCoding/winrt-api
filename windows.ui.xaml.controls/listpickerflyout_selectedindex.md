# Windows.UI.Xaml.Controls.ListPickerFlyout.SelectedIndex

## -description
Gets or sets the index of the selected item.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ListPickerFlyout.SelectedIndex](/windows/winui/api/microsoft.ui.xaml.controls.listpickerflyout.selectedindex) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The index of the selected item. The default is -1.
