# Windows.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelDown

## -description
Scrolls down within content after a user clicks the wheel button on a mouse.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollContentPresenter.MouseWheelDown](/windows/winui/api/microsoft.ui.xaml.controls.scrollcontentpresenter.mousewheeldown) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
