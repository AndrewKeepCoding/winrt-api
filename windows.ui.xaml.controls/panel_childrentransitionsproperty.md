# Windows.UI.Xaml.Controls.Panel.ChildrenTransitionsProperty

## -description
Identifies the [ChildrenTransitions](panel_childrentransitions.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Panel.ChildrenTransitionsProperty](/windows/winui/api/microsoft.ui.xaml.controls.panel.childrentransitionsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ChildrenTransitions](panel_childrentransitions.md) dependency property.
