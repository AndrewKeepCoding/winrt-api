# Windows.UI.Xaml.Controls.AppBarElementContainer.DynamicOverflowOrder

## -description

Gets or sets the order in which this item is moved to the [CommandBar](commandbar.md) overflow menu.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.AppBarElementContainer.DynamicOverflowOrder](/windows/winui/api/microsoft.ui.xaml.controls.appbarelementcontainer.dynamicoverfloworder) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
