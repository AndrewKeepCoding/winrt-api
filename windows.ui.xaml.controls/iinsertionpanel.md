# Windows.UI.Xaml.Controls.IInsertionPanel

## -description
Provides methods to let an item be inserted between other items in a drag-and-drop operation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.IInsertionPanel](/windows/winui/api/microsoft.ui.xaml.controls.iinsertionpanel) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
