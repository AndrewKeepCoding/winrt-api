# Windows.UI.Xaml.Controls.Control.ForegroundProperty

## -description
Identifies the [Foreground](control_foreground.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.ForegroundProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.foregroundproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Foreground](control_foreground.md) dependency property.
