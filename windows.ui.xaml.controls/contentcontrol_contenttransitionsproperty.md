# Windows.UI.Xaml.Controls.ContentControl.ContentTransitionsProperty

## -description
Identifies the [ContentTransitions](contentcontrol_contenttransitions.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentControl.ContentTransitionsProperty](/windows/winui/api/microsoft.ui.xaml.controls.contentcontrol.contenttransitionsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ContentTransitions](contentcontrol_contenttransitions.md) dependency property.
