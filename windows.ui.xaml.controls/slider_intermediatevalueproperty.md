# Windows.UI.Xaml.Controls.Slider.IntermediateValueProperty

## -description
Identifies the [IntermediateValue](slider_intermediatevalue.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Slider.IntermediateValueProperty](/windows/winui/api/microsoft.ui.xaml.controls.slider.intermediatevalueproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IntermediateValue](slider_intermediatevalue.md) dependency property.
