# Windows.UI.Xaml.Controls.PasswordBox.SelectAll

## -description
Selects all the characters in the [PasswordBox](passwordbox.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.SelectAll](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.selectall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
