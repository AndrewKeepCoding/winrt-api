# Windows.UI.Xaml.Controls.ContentDialog.CloseButtonCommandProperty

## -description
Gets the identifier of the CloseButtonCommand dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.CloseButtonCommandProperty](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.closebuttoncommandproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier of the CloseButtonCommand dependency property.
