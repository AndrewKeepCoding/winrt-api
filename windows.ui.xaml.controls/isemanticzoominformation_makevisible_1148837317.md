# Windows.UI.Xaml.Controls.ISemanticZoomInformation.MakeVisible

## -description
Forces content in the view to scroll until the item specified by [SemanticZoomLocation](semanticzoomlocation.md) is visible. Also focuses that item if found.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.MakeVisible](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.makevisible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param item
The item in the view to scroll to.
