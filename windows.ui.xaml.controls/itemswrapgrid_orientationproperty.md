# Windows.UI.Xaml.Controls.ItemsWrapGrid.OrientationProperty

## -description
Identifies the [Orientation](itemswrapgrid_orientation.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ItemsWrapGrid.OrientationProperty](/windows/winui/api/microsoft.ui.xaml.controls.itemswrapgrid.orientationproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Orientation](itemswrapgrid_orientation.md) dependency property.
