# Windows.UI.Xaml.Controls.ContentDialogClosedEventArgs.Result

## -description
Gets the [ContentDialogResult](contentdialogresult.md) of the button click event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialogClosedEventArgs.Result](/windows/winui/api/microsoft.ui.xaml.controls.contentdialogclosedeventargs.result) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The result of the button click event.
