# Windows.UI.Xaml.Controls.HubSectionHeaderClickEventArgs.Section

## -description
Gets the [HubSection](hubsection.md) for the header that was clicked.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.HubSectionHeaderClickEventArgs.Section](/windows/winui/api/microsoft.ui.xaml.controls.hubsectionheaderclickeventargs.section) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The [HubSection](hubsection.md) for the header that was clicked.
