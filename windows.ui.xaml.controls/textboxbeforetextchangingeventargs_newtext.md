# Windows.UI.Xaml.Controls.TextBoxBeforeTextChangingEventArgs.NewText

## -description

Gets the new text that is entered into the text box.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBoxBeforeTextChangingEventArgs.NewText](/windows/winui/api/microsoft.ui.xaml.controls.textboxbeforetextchangingeventargs.newtext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The new text value that is entered into the text box.
