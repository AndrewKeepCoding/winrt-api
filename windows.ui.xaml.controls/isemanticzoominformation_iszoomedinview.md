# Windows.UI.Xaml.Controls.ISemanticZoomInformation.IsZoomedInView

## -description
Gets or sets a value that indicates whether the implementing view is the semantically more complete zoomed-in view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.IsZoomedInView](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.iszoomedinview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the implementing view is the zoomed-in view; otherwise, **false**.
