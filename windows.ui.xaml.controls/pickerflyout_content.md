# Windows.UI.Xaml.Controls.PickerFlyout.Content

## -description
Gets or sets the content that is contained within the picker.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PickerFlyout.Content](/windows/winui/api/microsoft.ui.xaml.controls.pickerflyout.content) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The content of the user control.
