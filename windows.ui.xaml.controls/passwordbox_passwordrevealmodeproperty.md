# Windows.UI.Xaml.Controls.PasswordBox.PasswordRevealModeProperty

## -description
Identifies the [PasswordRevealMode](passwordbox_passwordrevealmode.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PasswordBox.PasswordRevealModeProperty](/windows/winui/api/microsoft.ui.xaml.controls.passwordbox.passwordrevealmodeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [PasswordRevealMode](passwordbox_passwordrevealmode.md) dependency property.
