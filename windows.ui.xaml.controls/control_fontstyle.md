# Windows.UI.Xaml.Controls.Control.FontStyle

## -description
Gets or sets the style in which the text is rendered.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.FontStyle](/windows/winui/api/microsoft.ui.xaml.controls.control.fontstyle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<control FontStyle="fontStyleMemberName" />
```


## -xaml-values
<dl><dt>fontStyleMemberName</dt><dd>fontStyleMemberNameA FontStyle named constant, such as Normal or Italic.</dd>
</dl>
## -property-value
One of the values that specifies the style in which the text is rendered. The default is **Normal**.
