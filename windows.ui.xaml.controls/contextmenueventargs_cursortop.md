# Windows.UI.Xaml.Controls.ContextMenuEventArgs.CursorTop

## -description
Gets the pixel offset of the text cursor vertical position.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContextMenuEventArgs.CursorTop](/windows/winui/api/microsoft.ui.xaml.controls.contextmenueventargs.cursortop) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
