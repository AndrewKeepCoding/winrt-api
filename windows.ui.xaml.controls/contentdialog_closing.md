# Windows.UI.Xaml.Controls.ContentDialog.Closing

## -description
Occurs after the dialog starts to close, but before it is closed and before the [Closed](contentdialog_closed.md) event occurs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.Closing](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.closing) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
