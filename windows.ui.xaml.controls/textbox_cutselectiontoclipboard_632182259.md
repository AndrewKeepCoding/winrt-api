# Windows.UI.Xaml.Controls.TextBox.CutSelectionToClipboard

## -description

Moves the selected content to the Windows clipboard and removes it from the text control.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.CutSelectionToClipboard](/windows/winui/api/microsoft.ui.xaml.controls.textbox.cutselectiontoclipboard) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
