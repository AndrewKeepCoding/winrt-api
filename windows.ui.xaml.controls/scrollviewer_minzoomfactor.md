# Windows.UI.Xaml.Controls.ScrollViewer.MinZoomFactor

## -description
Gets or sets a value that indicates the minimum permitted run-time value of [ZoomFactor](scrollviewer_zoomfactor.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.MinZoomFactor](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.minzoomfactor) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<ScrollViewer MinZoomFactor="single" />
```


## -property-value
The minimum permitted run-time value of [ZoomFactor](scrollviewer_zoomfactor.md). The default is 0.1.
