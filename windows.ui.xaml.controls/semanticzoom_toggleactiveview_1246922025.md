# Windows.UI.Xaml.Controls.SemanticZoom.ToggleActiveView

## -description
Changes from the current active view to the other possible view. For example, if [IsZoomedInViewActive](semanticzoom_iszoomedinviewactive.md) is true, calling this method changes to zoomed-out view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.SemanticZoom.ToggleActiveView](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoom.toggleactiveview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
