# Windows.UI.Xaml.Controls.UIElementCollection.GetView

## -description
Gets an immutable view into the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.UIElementCollection.GetView](/windows/winui/api/microsoft.ui.xaml.controls.uielementcollection.getview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
An object representing the immutable collection view. This object implements [IVectorView<T>](../windows.foundation.collections/ivectorview_1.md) with a [UIElement](../windows.ui.xaml/uielement.md) constraint.
