# Orientation

## -description
Defines constants that specify the different orientations that a control or layout can have.

Equivalent **WinUI 2 API for UWP**: [Orientation](/windows/winui/api/microsoft.ui.xaml.controls.orientation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
