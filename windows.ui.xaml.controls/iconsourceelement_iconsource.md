# Windows.UI.Xaml.Controls.IconSourceElement.IconSource

## -description

Gets or sets the IconSource used as the icon content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.IconSourceElement.IconSource](/windows/winui/api/microsoft.ui.xaml.controls.iconsourceelement.iconsource) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The IconSource used as the icon content. The default is **null**.
