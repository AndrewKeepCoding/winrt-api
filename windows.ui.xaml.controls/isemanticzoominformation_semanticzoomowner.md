# Windows.UI.Xaml.Controls.ISemanticZoomInformation.SemanticZoomOwner

## -description
Gets or sets the [SemanticZoom](semanticzoom.md) owner that hosts the implementing view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.SemanticZoomOwner](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.semanticzoomowner) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The [SemanticZoom](semanticzoom.md) that hosts this view.
