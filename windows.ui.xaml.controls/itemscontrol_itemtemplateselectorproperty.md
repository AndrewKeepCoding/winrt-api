# Windows.UI.Xaml.Controls.ItemsControl.ItemTemplateSelectorProperty

## -description
Identifies the [ItemTemplateSelector](itemscontrol_itemtemplateselector.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ItemsControl.ItemTemplateSelectorProperty](/windows/winui/api/microsoft.ui.xaml.controls.itemscontrol.itemtemplateselectorproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ItemTemplateSelector](itemscontrol_itemtemplateselector.md) dependency property.
