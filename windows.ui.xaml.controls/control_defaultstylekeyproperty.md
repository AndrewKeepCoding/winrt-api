# Windows.UI.Xaml.Controls.Control.DefaultStyleKeyProperty

## -description
Identifies the [DefaultStyleKey](control_defaultstylekey.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.DefaultStyleKeyProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.defaultstylekeyproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [DefaultStyleKey](control_defaultstylekey.md) dependency property.
