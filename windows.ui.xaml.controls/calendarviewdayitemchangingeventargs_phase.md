# Windows.UI.Xaml.Controls.CalendarViewDayItemChangingEventArgs.Phase

## -description
Gets the number of times this container and day item pair has been called.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CalendarViewDayItemChangingEventArgs.Phase](/windows/winui/api/microsoft.ui.xaml.controls.calendarviewdayitemchangingeventargs.phase) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The number of times this container and day item pair has been called.
