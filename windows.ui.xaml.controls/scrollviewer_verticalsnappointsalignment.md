# Windows.UI.Xaml.Controls.ScrollViewer.VerticalSnapPointsAlignment

## -description
Gets or sets a value that indicates how the existing snap points are vertically aligned versus the initial viewport.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.VerticalSnapPointsAlignment](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.verticalsnappointsalignment) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<ScrollViewer VerticalSnapPointsAlignment="snapPointsAlignmentMemberName" />
```


## -property-value
A value of the enumeration.
