# Windows.UI.Xaml.Controls.ScrollViewer.DirectManipulationStarted

## -description
Occurs when any direct manipulation of the [ScrollViewer](scrollviewer.md) begins.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.DirectManipulationStarted](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.directmanipulationstarted) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
