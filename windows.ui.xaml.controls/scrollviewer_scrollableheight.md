# Windows.UI.Xaml.Controls.ScrollViewer.ScrollableHeight

## -description
Gets a value that represents the vertical size of the area that can be scrolled; the difference between the height of the extent and the height of the viewport.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.ScrollableHeight](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.scrollableheight) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The vertical size of the area that can be scrolled. This property has no default value.
