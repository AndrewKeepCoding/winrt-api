# Windows.UI.Xaml.Controls.RichTextBlock.TextIndentProperty

## -description
Identifies the [TextIndent](richtextblock_textindent.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlock.TextIndentProperty](/windows/winui/api/microsoft.ui.xaml.controls.richtextblock.textindentproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier of the [TextIndent](richtextblock_textindent.md) dependency property.
