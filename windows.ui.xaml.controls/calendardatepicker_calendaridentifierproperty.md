# Windows.UI.Xaml.Controls.CalendarDatePicker.CalendarIdentifierProperty

## -description
Identifies the [CalendarIdentifier](calendardatepicker_calendaridentifier.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CalendarDatePicker.CalendarIdentifierProperty](/windows/winui/api/microsoft.ui.xaml.controls.calendardatepicker.calendaridentifierproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [CalendarIdentifier](calendardatepicker_calendaridentifier.md) dependency property.
