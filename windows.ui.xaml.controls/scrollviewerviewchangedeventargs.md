# Windows.UI.Xaml.Controls.ScrollViewerViewChangedEventArgs

## -description
Provides event data for the [ViewChanged](scrollviewer_viewchanged.md) event and similar control-specific events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewerViewChangedEventArgs](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewerviewchangedeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
