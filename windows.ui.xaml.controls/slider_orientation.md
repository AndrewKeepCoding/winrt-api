# Windows.UI.Xaml.Controls.Slider.Orientation

## -description
Gets or sets the orientation of a [Slider](slider.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Slider.Orientation](/windows/winui/api/microsoft.ui.xaml.controls.slider.orientation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<Slider Orientation="Horizontal"/>
-or-
<Slider Orientation="Vertical"/>
```


## -property-value
One of the [Orientation](orientation.md) values. The default is **Horizontal**.
