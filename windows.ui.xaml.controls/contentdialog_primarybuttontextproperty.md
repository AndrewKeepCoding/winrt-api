# Windows.UI.Xaml.Controls.ContentDialog.PrimaryButtonTextProperty

## -description
Gets the identifier of the PrimaryButtonText dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ContentDialog.PrimaryButtonTextProperty](/windows/winui/api/microsoft.ui.xaml.controls.contentdialog.primarybuttontextproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier of the PrimaryButtonText dependency property.
