# Windows.UI.Xaml.Controls.Image.StretchProperty

## -description
Identifies the [Stretch](image_stretch.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Image.StretchProperty](/windows/winui/api/microsoft.ui.xaml.controls.image.stretchproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Stretch](image_stretch.md)  dependency property.
