# Windows.UI.Xaml.Controls.CalendarDatePicker.MinDateProperty

## -description
Identifies the [MinDate](calendardatepicker_mindate.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.CalendarDatePicker.MinDateProperty](/windows/winui/api/microsoft.ui.xaml.controls.calendardatepicker.mindateproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MinDate](calendardatepicker_mindate.md) dependency property.
