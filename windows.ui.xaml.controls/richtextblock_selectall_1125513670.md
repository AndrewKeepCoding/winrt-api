# Windows.UI.Xaml.Controls.RichTextBlock.SelectAll

## -description
Selects the entire contents in the [RichTextBlock](richtextblock.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.RichTextBlock.SelectAll](/windows/winui/api/microsoft.ui.xaml.controls.richtextblock.selectall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
