# Windows.UI.Xaml.Controls.ChoosingItemContainerEventArgs.IsContainerPrepared

## -description
Gets or sets a value that indicates whether the container is ready for use.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ChoosingItemContainerEventArgs.IsContainerPrepared](/windows/winui/api/microsoft.ui.xaml.controls.choosingitemcontainereventargs.iscontainerprepared) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the container is ready for use; otherwise, **false**.
