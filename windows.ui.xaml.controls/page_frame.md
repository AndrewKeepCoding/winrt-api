# Windows.UI.Xaml.Controls.Page.Frame

## -description
Gets the controlling [Frame](frame.md) for the [Page](page.md) content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Page.Frame](/windows/winui/api/microsoft.ui.xaml.controls.page.frame) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The controlling [Frame](frame.md) for the [Page](page.md) content.
