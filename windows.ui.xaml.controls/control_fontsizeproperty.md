# Windows.UI.Xaml.Controls.Control.FontSizeProperty

## -description
Identifies the [FontSize](control_fontsize.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Control.FontSizeProperty](/windows/winui/api/microsoft.ui.xaml.controls.control.fontsizeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [FontSize](control_fontsize.md) dependency property.
