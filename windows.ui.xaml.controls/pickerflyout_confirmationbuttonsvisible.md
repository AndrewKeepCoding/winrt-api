# Windows.UI.Xaml.Controls.PickerFlyout.ConfirmationButtonsVisible

## -description
Gets or sets whether the confirmation buttons are visible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PickerFlyout.ConfirmationButtonsVisible](/windows/winui/api/microsoft.ui.xaml.controls.pickerflyout.confirmationbuttonsvisible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
True of the confirmation buttons are visible; Otherwise, false.
