# Windows.UI.Xaml.Controls.TextBox.SelectedText

## -description
Gets or sets the content of the current selection in the text box.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.TextBox.SelectedText](/windows/winui/api/microsoft.ui.xaml.controls.textbox.selectedtext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<TextBoxÂ SelectedText="string"/>
```


## -property-value
The currently selected text in the text box. If no text is selected, the value is [String.Empty](/dotnet/api/system.string.empty?view=dotnet-uwp-10.0&preserve-view=true).
