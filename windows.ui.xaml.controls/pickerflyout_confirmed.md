# Windows.UI.Xaml.Controls.PickerFlyout.Confirmed

## -description
Occurs when the user has tapped a confirmation button to confirm the selection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.PickerFlyout.Confirmed](/windows/winui/api/microsoft.ui.xaml.controls.pickerflyout.confirmed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
