# Windows.UI.Xaml.Controls.ScrollViewer.ScrollableWidth

## -description
Gets a value that represents the horizontal size of the area that can be scrolled; the difference between the width of the extent and the width of the viewport.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ScrollViewer.ScrollableWidth](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.scrollablewidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The horizontal size of the area that can be scrolled. This property has no default value.
