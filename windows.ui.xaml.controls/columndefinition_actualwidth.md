# Windows.UI.Xaml.Controls.ColumnDefinition.ActualWidth

## -description
Gets a value that represents the actual calculated width of a [ColumnDefinition](columndefinition.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ColumnDefinition.ActualWidth](/windows/winui/api/microsoft.ui.xaml.controls.columndefinition.actualwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A [Double](/dotnet/api/system.double?view=dotnet-uwp-10.0&preserve-view=true) that represents the actual calculated width in pixels. The default is 0.
