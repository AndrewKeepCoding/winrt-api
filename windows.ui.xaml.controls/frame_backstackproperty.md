# Windows.UI.Xaml.Controls.Frame.BackStackProperty

## -description
Identifies the [BackStack](frame_backstack.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.BackStackProperty](/windows/winui/api/microsoft.ui.xaml.controls.frame.backstackproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [BackStack](frame_backstack.md) dependency property.
