# Windows.UI.Xaml.Controls.Frame.CanGoForward

## -description
Gets a value that indicates whether there is at least one entry in forward navigation history.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Frame.CanGoForward](/windows/winui/api/microsoft.ui.xaml.controls.frame.cangoforward) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if there is at least one entry in forward navigation history; **false** if there are no entries in forward navigation history or the [Frame](frame.md) does not own its own navigation history.
