# Windows.UI.Xaml.Controls.ISemanticZoomInformation.IsActiveView

## -description
Gets or sets a value that indicates whether the implementing view is the active view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.IsActiveView](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.isactiveview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the implementing view is the active view; otherwise, **false**.
