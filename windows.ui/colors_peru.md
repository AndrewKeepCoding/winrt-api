# Windows.UI.Colors.Peru

## -description

Gets the system-defined color that has an ARGB value of #FFCD853F.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Peru](/windows/winui/api/microsoft.ui.colors.peru) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFCD853F.
