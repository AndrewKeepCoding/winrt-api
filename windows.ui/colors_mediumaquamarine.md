# Windows.UI.Colors.MediumAquamarine

## -description

Gets the system-defined color that has an ARGB value of #FF66CDAA.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.MediumAquamarine](/windows/winui/api/microsoft.ui.colors.mediumaquamarine) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF66CDAA.
