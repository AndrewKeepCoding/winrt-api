# Windows.UI.Colors.Blue

## -description

Gets the system-defined color that has the ARGB value of #FF0000FF.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Blue](/windows/winui/api/microsoft.ui.colors.blue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FF0000FF.
