# Windows.UI.Colors.Orchid

## -description

Gets the system-defined color that has an ARGB value of #FFDA70D6.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Orchid](/windows/winui/api/microsoft.ui.colors.orchid) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFDA70D6.
