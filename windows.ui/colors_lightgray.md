# Windows.UI.Colors.LightGray

## -description

Gets the system-defined color that has the ARGB value of #FFD3D3D3.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.LightGray](/windows/winui/api/microsoft.ui.colors.lightgray) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FFD3D3D3.
