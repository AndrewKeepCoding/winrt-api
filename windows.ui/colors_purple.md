# Windows.UI.Colors.Purple

## -description

Gets the system-defined color that has the ARGB value of #FF800080.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Purple](/windows/winui/api/microsoft.ui.colors.purple) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FF800080.
