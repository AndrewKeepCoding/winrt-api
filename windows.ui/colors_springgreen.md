# Windows.UI.Colors.SpringGreen

## -description

Gets the system-defined color that has an ARGB value of #FF00FF7F.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.SpringGreen](/windows/winui/api/microsoft.ui.colors.springgreen) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF00FF7F.
