# Windows.UI.Colors.PowderBlue

## -description

Gets the system-defined color that has an ARGB value of #FFB0E0E6.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.PowderBlue](/windows/winui/api/microsoft.ui.colors.powderblue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFB0E0E6.
