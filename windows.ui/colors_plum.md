# Windows.UI.Colors.Plum

## -description

Gets the system-defined color that has an ARGB value of #FFDDA0DD.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Plum](/windows/winui/api/microsoft.ui.colors.plum) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFDDA0DD.
