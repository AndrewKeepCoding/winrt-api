# Windows.UI.Colors.LightSalmon

## -description

Gets the system-defined color that has an ARGB value of #FFFFA07A.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.LightSalmon](/windows/winui/api/microsoft.ui.colors.lightsalmon) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFFA07A.
