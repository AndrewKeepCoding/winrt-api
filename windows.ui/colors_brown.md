# Windows.UI.Colors.Brown

## -description

Gets the system-defined color that has the ARGB value of #FFA52A2A.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Brown](/windows/winui/api/microsoft.ui.colors.brown) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FFA52A2A.
