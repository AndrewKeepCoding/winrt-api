# Windows.UI.Colors.DarkGray

## -description

Gets the system-defined color that has the ARGB value of #FFA9A9A9.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.DarkGray](/windows/winui/api/microsoft.ui.colors.darkgray) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FFA9A9A9.
