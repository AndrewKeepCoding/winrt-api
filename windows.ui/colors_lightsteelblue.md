# Windows.UI.Colors.LightSteelBlue

## -description

Gets the system-defined color that has an ARGB value of #FFB0C4DE.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.LightSteelBlue](/windows/winui/api/microsoft.ui.colors.lightsteelblue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFB0C4DE.
