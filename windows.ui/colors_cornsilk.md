# Windows.UI.Colors.Cornsilk

## -description

Gets the system-defined color that has an ARGB value of #FFFFF8DC.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Cornsilk](/windows/winui/api/microsoft.ui.colors.cornsilk) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFFF8DC.
