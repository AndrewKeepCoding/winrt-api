# Windows.UI.Colors.LightSeaGreen

## -description

Gets the system-defined color that has an ARGB value of #FF20B2AA.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.LightSeaGreen](/windows/winui/api/microsoft.ui.colors.lightseagreen) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF20B2AA.
