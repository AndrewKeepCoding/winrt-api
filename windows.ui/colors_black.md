# Windows.UI.Colors.Black

## -description

Gets the system-defined color that has the ARGB value of #FF000000.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Black](/windows/winui/api/microsoft.ui.colors.black) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has the ARGB value of #FF000000.
