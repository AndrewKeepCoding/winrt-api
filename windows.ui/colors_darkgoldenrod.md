# Windows.UI.Colors.DarkGoldenrod

## -description

Gets the system-defined color that has an ARGB value of #FFB8860B.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.DarkGoldenrod](/windows/winui/api/microsoft.ui.colors.darkgoldenrod) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFB8860B.
