# Windows.UI.Colors.Moccasin

## -description

Gets the system-defined color that has an ARGB value of #FFFFE4B5.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Moccasin](/windows/winui/api/microsoft.ui.colors.moccasin) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFFE4B5.
