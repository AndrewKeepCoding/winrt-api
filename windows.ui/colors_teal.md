# Windows.UI.Colors.Teal

## -description

Gets the system-defined color that has an ARGB value of #FF008080.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Teal](/windows/winui/api/microsoft.ui.colors.teal) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF008080.
