# Windows.UI.Colors.GhostWhite

## -description

Gets the system-defined color that has an ARGB value of #FFF8F8FF.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.GhostWhite](/windows/winui/api/microsoft.ui.colors.ghostwhite) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFF8F8FF.
