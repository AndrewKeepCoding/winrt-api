# Windows.UI.Colors.BlueViolet

## -description

Gets the system-defined color that has an ARGB value of #FF8A2BE2.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.BlueViolet](/windows/winui/api/microsoft.ui.colors.blueviolet) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF8A2BE2.
