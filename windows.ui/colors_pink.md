# Windows.UI.Colors.Pink

## -description

Gets the system-defined color that has an ARGB value of #FFFFC0CB.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.Pink](/windows/winui/api/microsoft.ui.colors.pink) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFFC0CB.
