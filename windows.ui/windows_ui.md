# Windows.UI

## -description
Provides an app with access to core system functionality and run-time information about its UI.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI](/windows/winui/api/microsoft.ui) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
