# Windows.UI.Colors.DarkSlateGray

## -description

Gets the system-defined color that has an ARGB value of #FF2F4F4F.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.DarkSlateGray](/windows/winui/api/microsoft.ui.colors.darkslategray) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF2F4F4F.
