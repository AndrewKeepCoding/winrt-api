# Windows.UI.Colors.RoyalBlue

## -description

Gets the system-defined color that has an ARGB value of #FF4169E1.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.RoyalBlue](/windows/winui/api/microsoft.ui.colors.royalblue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF4169E1.
