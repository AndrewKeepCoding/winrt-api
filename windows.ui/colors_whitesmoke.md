# Windows.UI.Colors.WhiteSmoke

## -description

Gets the system-defined color that has an ARGB value of #FFF5F5F5.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.WhiteSmoke](/windows/winui/api/microsoft.ui.colors.whitesmoke) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFF5F5F5.
