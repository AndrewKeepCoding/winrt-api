# Windows.UI.Colors.SeaGreen

## -description

Gets the system-defined color that has an ARGB value of #FF2E8B57.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.SeaGreen](/windows/winui/api/microsoft.ui.colors.seagreen) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF2E8B57.
