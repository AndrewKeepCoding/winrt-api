# Windows.UI.Colors.SlateBlue

## -description

Gets the system-defined color that has an ARGB value of #FF6A5ACD.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.SlateBlue](/windows/winui/api/microsoft.ui.colors.slateblue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF6A5ACD.
