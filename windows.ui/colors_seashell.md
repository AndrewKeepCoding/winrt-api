# Windows.UI.Colors.SeaShell

## -description

Gets the system-defined color that has an ARGB value of #FFFFF5EE.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.SeaShell](/windows/winui/api/microsoft.ui.colors.seashell) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFFF5EE.
