# Windows.UI.Colors.AntiqueWhite

## -description

Gets the system-defined color that has an ARGB value of #FFFAEBD7.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.AntiqueWhite](/windows/winui/api/microsoft.ui.colors.antiquewhite) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFAEBD7.
