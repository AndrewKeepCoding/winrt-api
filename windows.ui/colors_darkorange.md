# Windows.UI.Colors.DarkOrange

## -description

Gets the system-defined color that has an ARGB value of #FFFF8C00.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.DarkOrange](/windows/winui/api/microsoft.ui.colors.darkorange) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FFFF8C00.
