# Windows.UI.Colors.LawnGreen

## -description

Gets the system-defined color that has an ARGB value of #FF7CFC00.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Colors.LawnGreen](/windows/winui/api/microsoft.ui.colors.lawngreen) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The system-defined color that has an ARGB value of #FF7CFC00.
