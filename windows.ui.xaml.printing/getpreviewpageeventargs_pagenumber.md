# Windows.UI.Xaml.Printing.GetPreviewPageEventArgs.PageNumber

## -description
Gets the page number of the potentially repaginated page involved in the preview.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Printing.GetPreviewPageEventArgs.PageNumber](/windows/winui/api/microsoft.ui.xaml.printing.getpreviewpageeventargs.pagenumber) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The page number of the potentially repaginated page.
