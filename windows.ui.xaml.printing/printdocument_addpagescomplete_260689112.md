# Windows.UI.Xaml.Printing.PrintDocument.AddPagesComplete

## -description
Indicates that the application will not add more pages to the print list, and that the print list is ready to be released.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Printing.PrintDocument.AddPagesComplete](/windows/winui/api/microsoft.ui.xaml.printing.printdocument.addpagescomplete) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
