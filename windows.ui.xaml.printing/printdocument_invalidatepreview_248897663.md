# Windows.UI.Xaml.Printing.PrintDocument.InvalidatePreview

## -description
References the low-level print preview dialog API and invalidates the current print preview.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Printing.PrintDocument.InvalidatePreview](/windows/winui/api/microsoft.ui.xaml.printing.printdocument.invalidatepreview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
