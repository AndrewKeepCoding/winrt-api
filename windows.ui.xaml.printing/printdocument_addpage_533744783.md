# Windows.UI.Xaml.Printing.PrintDocument.AddPage

## -description
Adds a XAML root visual element to the print list.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Printing.PrintDocument.AddPage](/windows/winui/api/microsoft.ui.xaml.printing.printdocument.addpage) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param pageVisual
The root visual element of the page to add to the print list.
