# DeviceCommand

## -description

Commands that can be executed on a GIP headset.

## -enum-fields

### -field Reset:0

Resets the headset.

## -remarks

> [!CAUTION]
> To avoid damaging devices, the headset control APIs should only be used with hardware you have developed.
