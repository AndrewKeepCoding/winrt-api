# Windows.UI.Xaml.Interop

## -description
Provides basic types for cross-framework integration and interoperation. Notably, this includes interfaces and support types that enable binding patterns.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Interop](/windows/winui/api/microsoft.ui.xaml.interop) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
