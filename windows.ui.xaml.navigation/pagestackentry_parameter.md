# Windows.UI.Xaml.Navigation.PageStackEntry.Parameter

## -description
Gets the navigation parameter associated with this navigation entry.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Navigation.PageStackEntry.Parameter](/windows/winui/api/microsoft.ui.xaml.navigation.pagestackentry.parameter) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The navigation parameter.
