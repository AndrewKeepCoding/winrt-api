# Windows.UI.Xaml.Navigation.NavigationEventArgs.NavigationMode

## -description
Gets a value that indicates the direction of movement during navigation

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Navigation.NavigationEventArgs.NavigationMode](/windows/winui/api/microsoft.ui.xaml.navigation.navigationeventargs.navigationmode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
