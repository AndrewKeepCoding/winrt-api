# Windows.UI.Xaml.Navigation.NavigationEventArgs.Content

## -description
Gets the root node of the target page's content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Navigation.NavigationEventArgs.Content](/windows/winui/api/microsoft.ui.xaml.navigation.navigationeventargs.content) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The root node of the target page's content.
