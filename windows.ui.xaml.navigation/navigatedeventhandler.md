# Windows.UI.Xaml.Navigation.NavigatedEventHandler

## -description
Represents the method that will handle the [Navigated](../windows.ui.xaml.controls/frame_navigated.md) event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Navigation.NavigatedEventHandler](/windows/winui/api/microsoft.ui.xaml.navigation.navigatedeventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The object where the handler is attached.

### -param e
Event data for the event.
