# Windows.UI.Xaml.Navigation.NavigatingCancelEventArgs

## -description
Provides data for the [OnNavigatingFrom](../windows.ui.xaml.controls/page_onnavigatingfrom_425696585.md) callback that can be used to cancel a navigation request from origination.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Navigation.NavigatingCancelEventArgs](/windows/winui/api/microsoft.ui.xaml.navigation.navigatingcanceleventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
