# Windows.UI.Shell.FocusSessionManager.GetSession(System.String)

<!--
public Windows.UI.Shell.FocusSession GetSession (string id);

## -description

Gets a [FocusSession](focussession.md) object representing the focus session with the specified [Id](focussession_id.md).

## -parameters

### -param id

The unique identifier for a focus session.

## -returns

The requested [FocusSession](focussession.md) object if it exists; otherwise, null.

## -remarks


This method is a limited access feature which require specific approval to be used in an application. For more information, see [LimitedAccessFeatures](xref:Windows.ApplicationModel.LimitedAccessFeatures).
