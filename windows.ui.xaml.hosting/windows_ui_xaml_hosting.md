# Windows.UI.Xaml.Hosting

## -description
Provides services that are relevant for XAML design surfaces that are hosted in a larger application.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Hosting](/windows/winui/api/microsoft.ui.xaml.hosting) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
