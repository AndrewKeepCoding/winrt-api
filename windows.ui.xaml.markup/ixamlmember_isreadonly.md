# Windows.UI.Xaml.Markup.IXamlMember.IsReadOnly

## -description
Gets whether the XAML member is read-only in its backing implementation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Markup.IXamlMember.IsReadOnly](/windows/winui/api/microsoft.ui.xaml.markup.ixamlmember.isreadonly) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the backing member is read-only; otherwise, **false**.
