# Windows.UI.Xaml.Markup.IXamlType.ContentProperty

## -description
Gets the [IXamlMember](ixamlmember.md) information for the XAML content property of this [IXamlType](ixamltype.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Markup.IXamlType.ContentProperty](/windows/winui/api/microsoft.ui.xaml.markup.ixamltype.contentproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
[IXamlMember](ixamlmember.md) information for the XAML content property of the [IXamlType](ixamltype.md). May be null if no XAML content property exists.
