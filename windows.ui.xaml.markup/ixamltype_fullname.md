# Windows.UI.Xaml.Markup.IXamlType.FullName

## -description
Gets the full class name of the underlying type.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Markup.IXamlType.FullName](/windows/winui/api/microsoft.ui.xaml.markup.ixamltype.fullname) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The full class name of the underlying type.
