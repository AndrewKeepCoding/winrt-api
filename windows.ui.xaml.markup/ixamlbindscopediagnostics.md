# Windows.UI.Xaml.Markup.IXamlBindScopeDiagnostics

## -description

Provides methods that enable a debugger to disable generated binding code.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Markup.IXamlBindScopeDiagnostics](/windows/winui/api/microsoft.ui.xaml.markup.ixamlbindscopediagnostics) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks

This interface is for use in code that is generated by the XAML compiler.
