# Windows.UI.Xaml.Markup.IXamlType.IsConstructible

## -description
Gets a value that indicates whether this [IXamlType](ixamltype.md) represents a constructible type, as per the XAML definition.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Markup.IXamlType.IsConstructible](/windows/winui/api/microsoft.ui.xaml.markup.ixamltype.isconstructible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if this [IXamlType](ixamltype.md) represents a constructible type; otherwise, **false**.
