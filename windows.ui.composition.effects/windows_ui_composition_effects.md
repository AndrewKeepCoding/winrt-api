# Windows.UI.Composition.Effects

## -description
Contains classes that are specific to creating effects for User Interactions. These effects are specific to composition.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Composition.Effects](/windows/winui/api/microsoft.ui.composition.effects) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
