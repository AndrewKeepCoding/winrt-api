# Windows.UI.Xaml.Window.CoreWindow

## -description
Gets an internal *core* object for the application window.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Window.CoreWindow](/windows/winui/api/microsoft.ui.xaml.window.corewindow) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A **CoreWindow** object.
