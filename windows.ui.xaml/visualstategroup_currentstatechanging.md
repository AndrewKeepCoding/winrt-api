# Windows.UI.Xaml.VisualStateGroup.CurrentStateChanging

## -description
Occurs when a control begins changing into a different state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.VisualStateGroup.CurrentStateChanging](/windows/winui/api/microsoft.ui.xaml.visualstategroup.currentstatechanging) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
