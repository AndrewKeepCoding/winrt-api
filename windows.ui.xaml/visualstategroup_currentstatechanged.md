# Windows.UI.Xaml.VisualStateGroup.CurrentStateChanged

## -description
Occurs after a control changes into a different state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.VisualStateGroup.CurrentStateChanged](/windows/winui/api/microsoft.ui.xaml.visualstategroup.currentstatechanged) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
