# Windows.UI.Xaml.Window.Bounds

## -description
Gets a [Rect](../windows.foundation/rect.md) value containing the height and width of the application window in units of effective (view) pixels.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Window.Bounds](/windows/winui/api/microsoft.ui.xaml.window.bounds) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that reports the height and width of the application window.
