# Windows.UI.Xaml.ColorPaletteResources

## -description

Represents a specialized resource dictionary that contains color resources used by XAML elements.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.ColorPaletteResources](/windows/winui/api/microsoft.ui.xaml.colorpaletteresources) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
