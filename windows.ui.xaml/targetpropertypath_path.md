# Windows.UI.Xaml.TargetPropertyPath.Path

## -description
Gets or sets the path to the property on the target element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.TargetPropertyPath.Path](/windows/winui/api/microsoft.ui.xaml.targetpropertypath.path) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The path to the property on the target element.
