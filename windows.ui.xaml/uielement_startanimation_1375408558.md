# Windows.UI.Xaml.UIElement.StartAnimation

## -description
Begins the specified animation on the element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.StartAnimation](/windows/winui/api/microsoft.ui.xaml.uielement.startanimation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param animation

The animation to start.
