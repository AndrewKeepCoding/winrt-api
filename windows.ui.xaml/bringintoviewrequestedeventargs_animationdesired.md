# Windows.UI.Xaml.BringIntoViewRequestedEventArgs.AnimationDesired

## -description

Gets or sets a value that specifies whether the scrolling should be animated.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.BringIntoViewRequestedEventArgs.AnimationDesired](/windows/winui/api/microsoft.ui.xaml.bringintoviewrequestedeventargs.animationdesired) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

**true** to animate the scrolling; otherwise, **false**.
