# Windows.UI.Xaml.SetterBase.IsSealed

## -description
Gets a value that indicates whether this object is in an immutable state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SetterBase.IsSealed](/windows/winui/api/microsoft.ui.xaml.setterbase.issealed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if this object is in an immutable state; otherwise, **false**.
