# Windows.UI.Xaml.UIElement.ContextRequestedEvent

## -description

Gets the identifier for the ContextRequested routed event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.ContextRequestedEvent](/windows/winui/api/microsoft.ui.xaml.uielement.contextrequestedevent) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the ContextRequested routed event.
