# Windows.UI.Xaml.FrameworkViewSource

## -description
Creates views, specifically [FrameworkView](frameworkview.md) instances. This is infrastructure and does not need to be accessed in most app scenarios.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkViewSource](/windows/winui/api/microsoft.ui.xaml.frameworkviewsource) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
