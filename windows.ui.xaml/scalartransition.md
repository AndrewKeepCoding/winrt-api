# Windows.UI.Xaml.ScalarTransition

## -description
Provides the animated transition behavior when an element's Opacity or Rotation properties change.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.ScalarTransition](/windows/winui/api/microsoft.ui.xaml.scalartransition) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
