# Windows.UI.Xaml.SizeChangedEventArgs.PreviousSize

## -description
Gets the previous size of the object reporting the size change.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SizeChangedEventArgs.PreviousSize](/windows/winui/api/microsoft.ui.xaml.sizechangedeventargs.previoussize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The previous size.
