# Windows.UI.Xaml.UIElement.ContextFlyoutProperty

## -description
Identifies for the [ContextFlyout](uielement_contextflyout.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.ContextFlyoutProperty](/windows/winui/api/microsoft.ui.xaml.uielement.contextflyoutproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ContextFlyout](uielement_contextflyout.md) dependency property.
