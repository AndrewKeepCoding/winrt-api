# Windows.UI.Xaml.UIElement.KeyTipVerticalOffset

## -description
Gets or sets a value that indicates how far up or down the Key Tip is placed in relation to the UI element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.KeyTipVerticalOffset](/windows/winui/api/microsoft.ui.xaml.uielement.keytipverticaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<uiElement KeyTipVerticalOffset="double"/>
```

## -property-value
The number of pixels to move the Key Tip. Positive values move it down. Negative values move it up.
