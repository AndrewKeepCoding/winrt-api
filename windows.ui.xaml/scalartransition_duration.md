# Windows.UI.Xaml.ScalarTransition.Duration

## -description
Gets or sets the duration of the transition.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.ScalarTransition.Duration](/windows/winui/api/microsoft.ui.xaml.scalartransition.duration) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The duration of the transition.
