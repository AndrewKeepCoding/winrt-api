# Windows.UI.Xaml.SetterBaseCollection.GetView

## -description
Gets an immutable view into the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SetterBaseCollection.GetView](/windows/winui/api/microsoft.ui.xaml.setterbasecollection.getview) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
An object representing the immutable collection view.
