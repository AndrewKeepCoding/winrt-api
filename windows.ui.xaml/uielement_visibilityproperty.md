# Windows.UI.Xaml.UIElement.VisibilityProperty

## -description
Identifies the [Visibility](uielement_visibility.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.VisibilityProperty](/windows/winui/api/microsoft.ui.xaml.uielement.visibilityproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Visibility](uielement_visibility.md) dependency property.
