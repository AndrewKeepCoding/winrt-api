# Windows.UI.Xaml.SizeChangedEventArgs.NewSize

## -description
Gets the new size of the object reporting the size change.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SizeChangedEventArgs.NewSize](/windows/winui/api/microsoft.ui.xaml.sizechangedeventargs.newsize) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The new size. The [Size](../windows.foundation/size.md) structure contains the new height and width.
