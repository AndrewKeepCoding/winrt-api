# Windows.UI.Xaml.UIElement.NoFocusCandidateFoundEvent

## -description
Gets the identifier for the [NoFocusCandidateFound](uielement_nofocuscandidatefound.md) routed event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.NoFocusCandidateFoundEvent](/windows/winui/api/microsoft.ui.xaml.uielement.nofocuscandidatefoundevent) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [NoFocusCandidateFound](uielement_nofocuscandidatefound.md) routed event.
