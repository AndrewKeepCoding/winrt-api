# Windows.UI.Xaml.XamlRoot.Content

## -description

Gets the root element of the XAML element tree.  

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.XamlRoot.Content](/windows/winui/api/microsoft.ui.xaml.xamlroot.content) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The root element of the XAML element tree.
