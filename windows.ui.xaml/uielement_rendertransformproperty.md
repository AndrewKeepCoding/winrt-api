# Windows.UI.Xaml.UIElement.RenderTransformProperty

## -description
Identifies the [RenderTransform](uielement_rendertransform.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.RenderTransformProperty](/windows/winui/api/microsoft.ui.xaml.uielement.rendertransformproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [RenderTransform](uielement_rendertransform.md) dependency property.
