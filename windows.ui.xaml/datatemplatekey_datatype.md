# Windows.UI.Xaml.DataTemplateKey.DataType

## -description
Gets or sets the type for which the template is designed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.DataTemplateKey.DataType](/windows/winui/api/microsoft.ui.xaml.datatemplatekey.datatype) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The type of object that the template is used to display, or a string that specifies the XML tag name for the XML data that the template is used to display.
