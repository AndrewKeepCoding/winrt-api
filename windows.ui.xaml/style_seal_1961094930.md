# Windows.UI.Xaml.Style.Seal

## -description
Locks the style so that the [TargetType](style_targettype.md) property or any [Setter](setter.md) in the [Setters](style_setters.md) collection cannot be changed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Style.Seal](/windows/winui/api/microsoft.ui.xaml.style.seal) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
