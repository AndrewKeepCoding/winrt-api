# Windows.UI.Xaml.FrameworkElement.ActualTheme

## -description

Gets the UI theme that is currently used by the element, which might be different than the [RequestedTheme](frameworkelement_requestedtheme.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.ActualTheme](/windows/winui/api/microsoft.ui.xaml.frameworkelement.actualtheme) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A value of the enumeration, for example **Light**.
