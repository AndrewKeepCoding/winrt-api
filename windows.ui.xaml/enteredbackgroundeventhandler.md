# Windows.UI.Xaml.EnteredBackgroundEventHandler

## -description
Represents the method that will handle the [Application.EnteredBackground](application_enteredbackground.md) event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.EnteredBackgroundEventHandler](/windows/winui/api/microsoft.ui.xaml.enteredbackgroundeventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The object where the handler is attached.

### -param e
Event data for the event.
