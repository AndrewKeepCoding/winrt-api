# Windows.UI.Xaml.DispatcherTimer.IsEnabled

## -description
Gets a value that indicates whether the timer is running.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.DispatcherTimer.IsEnabled](/windows/winui/api/microsoft.ui.xaml.dispatchertimer.isenabled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the timer is enabled and running; otherwise, **false**.
