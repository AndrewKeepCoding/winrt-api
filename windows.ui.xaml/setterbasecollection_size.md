# Windows.UI.Xaml.SetterBaseCollection.Size

## -description
Gets the size (count) of the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SetterBaseCollection.Size](/windows/winui/api/microsoft.ui.xaml.setterbasecollection.size) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The count of items in the collection.

## -remarks
> [!NOTE]
> The equivalent Microsoft .NET API is [Count](/dotnet/api/system.collections.objectmodel.collection-1.count).
