# Windows.UI.Xaml.FrameworkView.SetWindow

## -description
Associates a [Window](window.md) with a view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkView.SetWindow](/windows/winui/api/microsoft.ui.xaml.frameworkview.setwindow) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param window
The [Window](window.md) to associate.
