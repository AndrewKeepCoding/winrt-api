# Windows.UI.Xaml.AdaptiveTrigger.MinWindowWidth

## -description
Gets or sets the minimum window width at which the [VisualState](visualstate.md) should be applied.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.AdaptiveTrigger.MinWindowWidth](/windows/winui/api/microsoft.ui.xaml.adaptivetrigger.minwindowwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<AdaptiveTrigger MinWindowWidth="double" .../>
```


## -property-value
The minimum window width (in effective pixels) at which the [VisualState](visualstate.md) should be applied.
