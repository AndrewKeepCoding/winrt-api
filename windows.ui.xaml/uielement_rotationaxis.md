# Windows.UI.Xaml.UIElement.RotationAxis

## -description
Gets or sets the axis to rotate the element around.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.RotationAxis](/windows/winui/api/microsoft.ui.xaml.uielement.rotationaxis) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The axis to rotate the element around.
