# Windows.UI.Xaml.UIElement.IsAccessKeyScope

## -description
Gets or sets a value that indicates whether an element defines its own access key scope.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.IsAccessKeyScope](/windows/winui/api/microsoft.ui.xaml.uielement.isaccesskeyscope) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if an element defines its own scope; otherwise, **false**.
