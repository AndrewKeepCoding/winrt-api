# Windows.UI.Xaml.StyleTypedPropertyAttribute

## -description
Represents an attribute that is applied to the class definition and determines the **TargetTypes** of the properties that are of type [Style](style.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.StyleTypedPropertyAttribute](/windows/winui/api/microsoft.ui.xaml.styletypedpropertyattribute) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
