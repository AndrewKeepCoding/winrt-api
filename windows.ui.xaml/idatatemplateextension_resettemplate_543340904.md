# Windows.UI.Xaml.IDataTemplateExtension.ResetTemplate

## -description
Disconnects any event listeners so the data template can be recycled.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.IDataTemplateExtension.ResetTemplate](/windows/winui/api/microsoft.ui.xaml.idatatemplateextension.resettemplate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
