# Windows.UI.Xaml.FrameworkElement.ActualThemeProperty

## -description

Identifies the ActualTheme dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.ActualThemeProperty](/windows/winui/api/microsoft.ui.xaml.frameworkelement.actualthemeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the ActualTheme dependency property.
