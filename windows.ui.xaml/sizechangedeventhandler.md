# Windows.UI.Xaml.SizeChangedEventHandler

## -description
Represents the method that will handle the [SizeChanged](frameworkelement_sizechanged.md) event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SizeChangedEventHandler](/windows/winui/api/microsoft.ui.xaml.sizechangedeventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The object where the event handler is attached.

### -param e
The event data.
