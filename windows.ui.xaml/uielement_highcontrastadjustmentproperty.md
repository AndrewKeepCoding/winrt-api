# Windows.UI.Xaml.UIElement.HighContrastAdjustmentProperty

## -description
Identifies the [HighContrastAdjustment](uielement_highcontrastadjustment.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.HighContrastAdjustmentProperty](/windows/winui/api/microsoft.ui.xaml.uielement.highcontrastadjustmentproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [HighContrastAdjustment](uielement_highcontrastadjustment.md) dependency property.
