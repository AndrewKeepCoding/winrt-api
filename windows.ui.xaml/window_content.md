# Windows.UI.Xaml.Window.Content

## -description
Gets or sets the visual root of an application window.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Window.Content](/windows/winui/api/microsoft.ui.xaml.window.content) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The visual root of an application window.
