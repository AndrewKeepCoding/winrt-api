# Windows.UI.Xaml.BringIntoViewOptions.AnimationDesired

## -description
Gets or sets a value that indicates whether to use animation when the element is brought into view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.BringIntoViewOptions.AnimationDesired](/windows/winui/api/microsoft.ui.xaml.bringintoviewoptions.animationdesired) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** to use animation; otherwise, **false**. The default is **true**.
