# Windows.UI.Xaml.Window.Compositor

## -description
Gets the [Compositor](../windows.ui.composition/compositor.md) for this window.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Window.Compositor](/windows/winui/api/microsoft.ui.xaml.window.compositor) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The [Compositor](../windows.ui.composition/compositor.md) for this window.
