# Windows.UI.Xaml.TriggerActionCollection.SetAt

## -description
Sets the value at the specified index to the [TriggerAction](triggeraction.md) value specified.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.TriggerActionCollection.SetAt](/windows/winui/api/microsoft.ui.xaml.triggeractioncollection.setat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index at which to set the value.

### -param value
The value to set.

## -remarks
> [!NOTE]
> The equivalent Microsoft .NET API is [Item](/dotnet/api/system.windows.triggeractioncollection.item) (the indexer).
