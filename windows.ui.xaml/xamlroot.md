# Windows.UI.Xaml.XamlRoot

## -description

Represents a tree of XAML content and information about the context in which it is hosted.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.XamlRoot](/windows/winui/api/microsoft.ui.xaml.xamlroot) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
