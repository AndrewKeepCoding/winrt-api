# Windows.UI.Xaml.UIElement.IsHitTestVisible

## -description
Gets or sets whether the contained area of this [UIElement](uielement.md) can return true values for hit testing.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.IsHitTestVisible](/windows/winui/api/microsoft.ui.xaml.uielement.ishittestvisible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<uiElement IsHitTestVisible="bool"/>
```


## -property-value
**true** if the contained area of this [UIElement](uielement.md) can be used for hit testing; otherwise, **false**. The default is **true**.
