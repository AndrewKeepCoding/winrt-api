# Windows.UI.Xaml.DragUIOverride.IsGlyphVisible

## -description
Gets or sets a value that indicates whether the glyph is shown.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.DragUIOverride.IsGlyphVisible](/windows/winui/api/microsoft.ui.xaml.draguioverride.isglyphvisible) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the glyph is shown; otherwise, **false**.
