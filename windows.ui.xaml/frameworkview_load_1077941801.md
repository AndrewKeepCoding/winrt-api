# Windows.UI.Xaml.FrameworkView.Load

## -description
Loads the view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkView.Load](/windows/winui/api/microsoft.ui.xaml.frameworkview.load) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param entryPoint
An identifier, such as a class name from the application code, that provides additional information about the view.
