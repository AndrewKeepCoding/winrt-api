# Windows.UI.Xaml.TriggerActionCollection.RemoveAt

## -description
Removes the item at the specified index.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.TriggerActionCollection.RemoveAt](/windows/winui/api/microsoft.ui.xaml.triggeractioncollection.removeat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index of the item to remove.

## -remarks
> [!NOTE]
> The equivalent Microsoft .NET  API is RemoveAt.
