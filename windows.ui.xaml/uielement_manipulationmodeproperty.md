# Windows.UI.Xaml.UIElement.ManipulationModeProperty

## -description
Identifies the [ManipulationMode](uielement_manipulationmode.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.ManipulationModeProperty](/windows/winui/api/microsoft.ui.xaml.uielement.manipulationmodeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ManipulationMode](uielement_manipulationmode.md) dependency property.
