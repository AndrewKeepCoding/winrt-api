# Windows.UI.Xaml.FrameworkElement.AllowFocusWhenDisabled

## -description
Gets or sets whether a disabled control can receive focus.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.AllowFocusWhenDisabled](/windows/winui/api/microsoft.ui.xaml.frameworkelement.allowfocuswhendisabled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if a disabled control can receive focus; otherwise, **false**.
