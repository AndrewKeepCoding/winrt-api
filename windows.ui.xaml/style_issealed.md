# Windows.UI.Xaml.Style.IsSealed

## -description
Gets a value that indicates whether the style is read-only and cannot be changed.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Style.IsSealed](/windows/winui/api/microsoft.ui.xaml.style.issealed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the style is read-only; otherwise, **false**.
