# Windows.UI.Xaml.UIElement.Rotation

## -description
Gets or sets the angle of clockwise rotation, in degrees. Rotates relative to the RotationAxis and the CenterPoint. Affects the rendering position of the element.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.Rotation](/windows/winui/api/microsoft.ui.xaml.uielement.rotation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The angle of clockwise rotation, in degrees.
