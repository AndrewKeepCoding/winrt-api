# Windows.UI.Xaml.MediaFailedRoutedEventArgs.ErrorTrace

## -description
Gets the trace information for a media failed event.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.MediaFailedRoutedEventArgs.ErrorTrace](/windows/winui/api/microsoft.ui.xaml.mediafailedroutedeventargs.errortrace) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The error trace for the failed media event.
