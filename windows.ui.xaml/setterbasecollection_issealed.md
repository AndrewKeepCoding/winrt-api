# Windows.UI.Xaml.SetterBaseCollection.IsSealed

## -description
Gets a value that indicates whether the collection is in a read-only state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.SetterBaseCollection.IsSealed](/windows/winui/api/microsoft.ui.xaml.setterbasecollection.issealed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if this object is in a read-only state and cannot be changed; otherwise, **false**.
