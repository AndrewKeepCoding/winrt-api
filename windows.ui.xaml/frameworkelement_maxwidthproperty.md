# Windows.UI.Xaml.FrameworkElement.MaxWidthProperty

## -description
Identifies the [MaxWidth](frameworkelement_maxwidth.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.MaxWidthProperty](/windows/winui/api/microsoft.ui.xaml.frameworkelement.maxwidthproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MaxWidth](frameworkelement_maxwidth.md)  dependency property.
