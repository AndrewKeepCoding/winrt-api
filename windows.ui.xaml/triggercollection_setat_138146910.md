# Windows.UI.Xaml.TriggerCollection.SetAt

## -description
Sets the value at the specified index to the [TriggerBase](triggerbase.md) value specified.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.TriggerCollection.SetAt](/windows/winui/api/microsoft.ui.xaml.triggercollection.setat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index at which to set the value.

### -param value
The value to set.

## -remarks
> [!NOTE]
> The equivalent Microsoft .NET API is [Item](/dotnet/api/system.collections.objectmodel.collection-1.item) (the indexer).
