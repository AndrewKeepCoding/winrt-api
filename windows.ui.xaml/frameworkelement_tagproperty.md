# Windows.UI.Xaml.FrameworkElement.TagProperty

## -description
Identifies the [Tag](frameworkelement_tag.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.TagProperty](/windows/winui/api/microsoft.ui.xaml.frameworkelement.tagproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Tag](frameworkelement_tag.md)  dependency property.
