# Windows.UI.Xaml.UIElement.ScaleTransition

## -description
Gets or sets the Vector3Transition that animates changes to the Scale property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.ScaleTransition](/windows/winui/api/microsoft.ui.xaml.uielement.scaletransition) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The Vector3Transition that animates changes to the Scale property.
