# Windows.UI.Xaml.FrameworkElement.MinWidthProperty

## -description
Identifies the [MinWidth](frameworkelement_minwidth.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.MinWidthProperty](/windows/winui/api/microsoft.ui.xaml.frameworkelement.minwidthproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [MinWidth](frameworkelement_minwidth.md)  dependency property.
