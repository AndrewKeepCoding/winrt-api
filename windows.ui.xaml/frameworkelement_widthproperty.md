# Windows.UI.Xaml.FrameworkElement.WidthProperty

## -description
Identifies the [Width](frameworkelement_width.md)  dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.FrameworkElement.WidthProperty](/windows/winui/api/microsoft.ui.xaml.frameworkelement.widthproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Width](frameworkelement_width.md)  dependency property.
