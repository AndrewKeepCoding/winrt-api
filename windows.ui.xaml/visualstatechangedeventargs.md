# Windows.UI.Xaml.VisualStateChangedEventArgs

## -description
Provides data for the [CurrentStateChanging](visualstategroup_currentstatechanging.md) and [CurrentStateChanged](visualstategroup_currentstatechanged.md) events.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.VisualStateChangedEventArgs](/windows/winui/api/microsoft.ui.xaml.visualstatechangedeventargs) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
