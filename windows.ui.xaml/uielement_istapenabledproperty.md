# Windows.UI.Xaml.UIElement.IsTapEnabledProperty

## -description
Identifies the [IsTapEnabled](uielement_istapenabled.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.UIElement.IsTapEnabledProperty](/windows/winui/api/microsoft.ui.xaml.uielement.istapenabledproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsTapEnabled](uielement_istapenabled.md) dependency property.
