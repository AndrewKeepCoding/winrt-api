# Windows.UI.Xaml.Controls.Primitives.CarouselPanel.AreVerticalSnapPointsRegular

## -description
Gets a value that indicates whether the vertical snap points for the [CarouselPanel](carouselpanel.md) are equidistant from each other.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.CarouselPanel.AreVerticalSnapPointsRegular](/windows/winui/api/microsoft.ui.xaml.controls.primitives.carouselpanel.areverticalsnappointsregular) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the vertical snap points for the [CarouselPanel](carouselpanel.md) are equidistant from each other; otherwise, **false**.
