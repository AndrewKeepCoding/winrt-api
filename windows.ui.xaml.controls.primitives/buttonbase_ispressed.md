# Windows.UI.Xaml.Controls.Primitives.ButtonBase.IsPressed

## -description
Gets a value that indicates whether a [ButtonBase](buttonbase.md) is currently in a pressed state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ButtonBase.IsPressed](/windows/winui/api/microsoft.ui.xaml.controls.primitives.buttonbase.ispressed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**True** if the [ButtonBase](buttonbase.md) is in a pressed state; otherwise **false**. The default is **false**.
