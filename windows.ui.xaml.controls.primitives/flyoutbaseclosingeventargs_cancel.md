# Windows.UI.Xaml.Controls.Primitives.FlyoutBaseClosingEventArgs.Cancel

## -description
Gets or sets a value that indicates whether the flyout should be prevented from closing.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.FlyoutBaseClosingEventArgs.Cancel](/windows/winui/api/microsoft.ui.xaml.controls.primitives.flyoutbaseclosingeventargs.cancel) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** to prevent the flyout from closing; otherwise, **false**.
