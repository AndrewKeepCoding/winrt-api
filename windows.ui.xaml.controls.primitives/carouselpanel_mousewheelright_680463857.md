# Windows.UI.Xaml.Controls.Primitives.CarouselPanel.MouseWheelRight

## -description
Scrolls content by one mousewheel click towards the right.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.CarouselPanel.MouseWheelRight](/windows/winui/api/microsoft.ui.xaml.controls.primitives.carouselpanel.mousewheelright) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -remarks
This implementation calls [SetHorizontalOffset](carouselpanel_sethorizontaloffset_1971679761.md) using a default value for "mousewheel click" of 48 pixels (3 lines) that comes from a default [ScrollViewer](../windows.ui.xaml.controls/scrollviewer.md).
