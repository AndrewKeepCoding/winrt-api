# Windows.UI.Xaml.Controls.Primitives.ScrollBar.IndicatorModeProperty

## -description
Identifies the [IndicatorMode](scrollbar_indicatormode.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ScrollBar.IndicatorModeProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.scrollbar.indicatormodeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IndicatorMode](scrollbar_indicatormode.md) dependency property.
