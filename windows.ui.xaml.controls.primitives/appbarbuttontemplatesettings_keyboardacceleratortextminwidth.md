# Windows.UI.Xaml.Controls.Primitives.AppBarButtonTemplateSettings.KeyboardAcceleratorTextMinWidth

## -description
Gets the minimum width allocated for the accelerator key tip of an [AppBarButton](../windows.ui.xaml.controls/appbarbutton.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.AppBarButtonTemplateSettings.KeyboardAcceleratorTextMinWidth](/windows/winui/api/microsoft.ui.xaml.controls.primitives.appbarbuttontemplatesettings.keyboardacceleratortextminwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A number that specifies the minimum width allocated for the accelerator key tip of an [AppBarButton](../windows.ui.xaml.controls/appbarbutton.md).
