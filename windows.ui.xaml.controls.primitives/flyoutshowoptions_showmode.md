# Windows.UI.Xaml.Controls.Primitives.FlyoutShowOptions.ShowMode

## -description

Gets or sets a value that indicates how the flyout behaves when opened.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.FlyoutShowOptions.ShowMode](/windows/winui/api/microsoft.ui.xaml.controls.primitives.flyoutshowoptions.showmode) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

A value of the enumeration that indicates how the flyout behaves when opened.
