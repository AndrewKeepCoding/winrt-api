# Windows.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.ViewportWidth

## -description
Gets the horizontal size of the viewport or content area.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.ViewportWidth](/windows/winui/api/microsoft.ui.xaml.controls.primitives.orientedvirtualizingpanel.viewportwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
