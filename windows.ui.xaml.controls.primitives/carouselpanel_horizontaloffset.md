# Windows.UI.Xaml.Controls.Primitives.CarouselPanel.HorizontalOffset

## -description
Gets the horizontal offset of the scrolled content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.CarouselPanel.HorizontalOffset](/windows/winui/api/microsoft.ui.xaml.controls.primitives.carouselpanel.horizontaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
