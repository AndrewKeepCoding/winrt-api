# Windows.UI.Xaml.Controls.Primitives.CarouselPanel.CanHorizontallyScroll

## -description
Not intended for general use. Gets or sets a value that indicates whether scrolling on the horizontal axis is possible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.CarouselPanel.CanHorizontallyScroll](/windows/winui/api/microsoft.ui.xaml.controls.primitives.carouselpanel.canhorizontallyscroll) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<CarouselPanel CanHorizontallyScroll="bool" />
```


## -property-value
**true** if scrolling is possible; otherwise, **false**.

## -remarks
> [!NOTE]
> This property is not intended for use in your code.
