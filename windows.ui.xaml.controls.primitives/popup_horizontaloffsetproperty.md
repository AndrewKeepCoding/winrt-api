# Windows.UI.Xaml.Controls.Primitives.Popup.HorizontalOffsetProperty

## -description
Gets the identifier for the [HorizontalOffset](popup_horizontaloffset.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Popup.HorizontalOffsetProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.popup.horizontaloffsetproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [HorizontalOffset](popup_horizontaloffset.md) dependency property.
