# Windows.UI.Xaml.Controls.Primitives.SelectorItem.IsSelected

## -description
Gets or sets a value that indicates whether the item is selected in a selector.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.SelectorItem.IsSelected](/windows/winui/api/microsoft.ui.xaml.controls.primitives.selectoritem.isselected) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<selectorItem IsSelected="bool" />
```


## -property-value
**true** if the item is selected; otherwise, **false**.
