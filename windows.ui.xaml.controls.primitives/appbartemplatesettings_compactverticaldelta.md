# Windows.UI.Xaml.Controls.Primitives.AppBarTemplateSettings.CompactVerticalDelta

## -description
Gets the vertical delta of the [AppBar](../windows.ui.xaml.controls/appbar.md) in the compact state.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.AppBarTemplateSettings.CompactVerticalDelta](/windows/winui/api/microsoft.ui.xaml.controls.primitives.appbartemplatesettings.compactverticaldelta) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The vertical delta of the [AppBar](../windows.ui.xaml.controls/appbar.md) in the compact state.
