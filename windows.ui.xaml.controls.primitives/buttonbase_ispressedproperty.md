# Windows.UI.Xaml.Controls.Primitives.ButtonBase.IsPressedProperty

## -description
Identifies the [IsPressed](buttonbase_ispressed.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ButtonBase.IsPressedProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.buttonbase.ispressedproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsPressed](buttonbase_ispressed.md) dependency property.
