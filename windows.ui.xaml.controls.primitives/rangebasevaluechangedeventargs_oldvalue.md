# Windows.UI.Xaml.Controls.Primitives.RangeBaseValueChangedEventArgs.OldValue

## -description
Gets the previous value of a range value property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.RangeBaseValueChangedEventArgs.OldValue](/windows/winui/api/microsoft.ui.xaml.controls.primitives.rangebasevaluechangedeventargs.oldvalue) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The previous value.
