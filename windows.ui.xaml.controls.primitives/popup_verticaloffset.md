# Windows.UI.Xaml.Controls.Primitives.Popup.VerticalOffset

## -description
Gets or sets the distance between the top of the application window and the top of the popup.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Popup.VerticalOffset](/windows/winui/api/microsoft.ui.xaml.controls.primitives.popup.verticaloffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
