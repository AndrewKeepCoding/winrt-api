# Windows.UI.Xaml.Controls.Primitives.Popup.Closed

## -description
Fires when the [IsOpen](popup_isopen.md) property is set to false.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Popup.Closed](/windows/winui/api/microsoft.ui.xaml.controls.primitives.popup.closed) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
