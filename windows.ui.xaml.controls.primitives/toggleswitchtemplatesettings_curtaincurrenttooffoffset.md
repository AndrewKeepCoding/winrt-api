# Windows.UI.Xaml.Controls.Primitives.ToggleSwitchTemplateSettings.CurtainCurrentToOffOffset

## -description
Gets the calculated value of the offset for the "curtain" element of the [ToggleSwitch](../windows.ui.xaml.controls/toggleswitch.md), calculated for an "Off" action and current switch position.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ToggleSwitchTemplateSettings.CurtainCurrentToOffOffset](/windows/winui/api/microsoft.ui.xaml.controls.primitives.toggleswitchtemplatesettings.curtaincurrenttooffoffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
