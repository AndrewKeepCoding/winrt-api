# Windows.UI.Xaml.Controls.Primitives.LoopingSelector.ItemTemplate

## -description
Gets or sets the [DataTemplate](../windows.ui.xaml/datatemplate.md) used to display each [LoopingSelectorItem](loopingselectoritem.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.LoopingSelector.ItemTemplate](/windows/winui/api/microsoft.ui.xaml.controls.primitives.loopingselector.itemtemplate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The template that specifies the visualization of the [LoopingSelectorItem](loopingselectoritem.md) items.
