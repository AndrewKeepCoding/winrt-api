# Windows.UI.Xaml.Controls.Primitives.ScrollEventHandler

## -description
Represents the method that will handle the [Scroll](scrollbar_scroll.md) event of a [ScrollBar](scrollbar.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ScrollEventHandler](/windows/winui/api/microsoft.ui.xaml.controls.primitives.scrolleventhandler) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param sender
The source of the event.

### -param e
A [ScrollEventArgs](scrolleventargs.md) that contains the event data.
