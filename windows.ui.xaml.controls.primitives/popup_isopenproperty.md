# Windows.UI.Xaml.Controls.Primitives.Popup.IsOpenProperty

## -description
Gets the identifier for the [IsOpen](popup_isopen.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Popup.IsOpenProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.popup.isopenproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [IsOpen](popup_isopen.md) dependency property.
