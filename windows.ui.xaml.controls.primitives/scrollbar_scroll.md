# Windows.UI.Xaml.Controls.Primitives.ScrollBar.Scroll

## -description
Occurs one or more times as content scrolls in a [ScrollBar](scrollbar.md) when the user moves the [Thumb](thumb.md) by using the mouse.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ScrollBar.Scroll](/windows/winui/api/microsoft.ui.xaml.controls.primitives.scrollbar.scroll) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
