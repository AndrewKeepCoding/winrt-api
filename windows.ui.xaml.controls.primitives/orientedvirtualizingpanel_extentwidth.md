# Windows.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.ExtentWidth

## -description
Gets the horizontal size of the panel extent.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.ExtentWidth](/windows/winui/api/microsoft.ui.xaml.controls.primitives.orientedvirtualizingpanel.extentwidth) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
