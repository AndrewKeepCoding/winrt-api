# Windows.UI.Xaml.Controls.Primitives.ToggleSwitchTemplateSettings.KnobOnToOffOffset

## -description
Gets the calculated value of the offset for the "knob" element of the [ToggleSwitch](../windows.ui.xaml.controls/toggleswitch.md), in "On" to "Off" direction.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ToggleSwitchTemplateSettings.KnobOnToOffOffset](/windows/winui/api/microsoft.ui.xaml.controls.primitives.toggleswitchtemplatesettings.knobontooffoffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value in pixels.
