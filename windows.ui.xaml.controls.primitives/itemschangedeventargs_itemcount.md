# Windows.UI.Xaml.Controls.Primitives.ItemsChangedEventArgs.ItemCount

## -description
Gets the number of items that were involved in the change.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ItemsChangedEventArgs.ItemCount](/windows/winui/api/microsoft.ui.xaml.controls.primitives.itemschangedeventargs.itemcount) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
Integer that represents the number of items involved in the change.
