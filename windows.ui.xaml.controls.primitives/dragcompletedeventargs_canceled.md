# Windows.UI.Xaml.Controls.Primitives.DragCompletedEventArgs.Canceled

## -description
Gets a value that indicates whether the drag operation was canceled.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.DragCompletedEventArgs.Canceled](/windows/winui/api/microsoft.ui.xaml.controls.primitives.dragcompletedeventargs.canceled) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the drag operation was canceled; otherwise, **false**.
