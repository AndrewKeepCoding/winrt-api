# Windows.UI.Xaml.Controls.Primitives.ScrollBar.ViewportSizeProperty

## -description
Identifies the [ViewportSize](scrollbar_viewportsize.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.ScrollBar.ViewportSizeProperty](/windows/winui/api/microsoft.ui.xaml.controls.primitives.scrollbar.viewportsizeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ViewportSize](scrollbar_viewportsize.md) dependency property.
