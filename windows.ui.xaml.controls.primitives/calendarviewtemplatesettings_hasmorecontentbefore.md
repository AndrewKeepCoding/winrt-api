# Windows.UI.Xaml.Controls.Primitives.CalendarViewTemplateSettings.HasMoreContentBefore

## -description
Gets a value that indicates whether the [CalendarView](../windows.ui.xaml.controls/calendarview.md) has more content before the displayed content.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.CalendarViewTemplateSettings.HasMoreContentBefore](/windows/winui/api/microsoft.ui.xaml.controls.primitives.calendarviewtemplatesettings.hasmorecontentbefore) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the [CalendarView](../windows.ui.xaml.controls/calendarview.md) has more content after the displayed content; otherwise, **false**.
