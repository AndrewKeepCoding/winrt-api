# Windows.UI.Xaml.Controls.Primitives.Thumb.IsDragging

## -description
Gets whether the [Thumb](thumb.md) control has focus and mouse capture.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Controls.Primitives.Thumb.IsDragging](/windows/winui/api/microsoft.ui.xaml.controls.primitives.thumb.isdragging) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
**true** if the [Thumb](thumb.md) control has focus and mouse capture; otherwise, **false**. The default is **false**.
