# Windows.Devices.Display.DisplayMonitor.FromInterfaceIdAsync

## -description
Asynchronously creates a **DisplayMonitor** object for the specified monitor interface identifier.

> [!NOTE]
> Most Win32 APIs that provide a monitor identifier provide the monitor interface identifier (and not the device identifier).

## -parameters
### -param deviceInterfaceId
The monitor interface identifier

## -returns
An asynchronous create operation.
