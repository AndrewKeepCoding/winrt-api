# Windows.UI.Xaml.Media.Animation.DrillInThemeAnimation

## -description
Represents a preconfigured animation that runs when a user navigates forward in a logical hierarchy, like from a master page to a detail page.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.DrillInThemeAnimation](/windows/winui/api/microsoft.ui.xaml.media.animation.drillinthemeanimation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
