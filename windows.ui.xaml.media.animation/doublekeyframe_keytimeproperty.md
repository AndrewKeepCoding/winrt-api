# Windows.UI.Xaml.Media.Animation.DoubleKeyFrame.KeyTimeProperty

## -description
Identifies the [KeyTime](colorkeyframe_keytime.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.DoubleKeyFrame.KeyTimeProperty](/windows/winui/api/microsoft.ui.xaml.media.animation.doublekeyframe.keytimeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [KeyTime](colorkeyframe_keytime.md) dependency property.
