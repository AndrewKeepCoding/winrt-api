# Windows.UI.Xaml.Media.Animation.SlideNavigationTransitionInfo.Effect

## -description
Gets or sets the type of animation effect to play during the slide transition.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.SlideNavigationTransitionInfo.Effect](/windows/winui/api/microsoft.ui.xaml.media.animation.slidenavigationtransitioninfo.effect) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The type of animation effect to play during the slide transition.
