# Windows.UI.Xaml.Media.Animation.TransitionCollection.First

## -description
Returns the iterator for iteration over the items in the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.TransitionCollection.First](/windows/winui/api/microsoft.ui.xaml.media.animation.transitioncollection.first) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The iterator object. The iterator's current position is at the 0-index position, or at the collection end if the collection is empty.
