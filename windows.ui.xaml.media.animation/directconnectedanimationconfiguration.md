# Windows.UI.Xaml.Media.Animation.DirectConnectedAnimationConfiguration

## -description
An object that configures the connected animation to play a quick linear animation between source and destination.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.DirectConnectedAnimationConfiguration](/windows/winui/api/microsoft.ui.xaml.media.animation.directconnectedanimationconfiguration) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
