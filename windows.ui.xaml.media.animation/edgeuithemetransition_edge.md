# Windows.UI.Xaml.Media.Animation.EdgeUIThemeTransition.Edge

## -description
Gets or sets the edge position to use for the transition.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.EdgeUIThemeTransition.Edge](/windows/winui/api/microsoft.ui.xaml.media.animation.edgeuithemetransition.edge) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<EdgeUIThemeTransition Edge="edgeTransitionLocationMemberName" .../>
```


## -xaml-values
<dl><dt>edgeTransitionLocationMemberName</dt><dd>edgeTransitionLocationMemberNameA constant name from the EdgeTransitionLocation enumeration, for example Bottom.</dd>
</dl>
## -property-value
A value of the enumeration. The default is **Top**.
