# Windows.UI.Xaml.Media.Animation.EntranceNavigationTransitionInfo

## -description
Specifies the animation to run when content appears on a [Page](../windows.ui.xaml.controls/page.md).

## -remarks
With Windows 10, version 1803, a Frame uses [**NavigationThemeTransition**](navigationthemetransition.md) to animate navigation between Pages. By default, the animation is *Page Refresh*, which is represented by the **EntranceNavigationTransitionInfo** class.
