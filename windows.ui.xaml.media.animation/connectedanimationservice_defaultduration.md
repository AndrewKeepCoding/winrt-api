# Windows.UI.Xaml.Media.Animation.ConnectedAnimationService.DefaultDuration

## -description
Gets or sets the default time that the animation runs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.ConnectedAnimationService.DefaultDuration](/windows/winui/api/microsoft.ui.xaml.media.animation.connectedanimationservice.defaultduration) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The default time that the animation runs.
