# Windows.UI.Xaml.Media.Animation.SplitCloseThemeAnimation.OpenedTarget

## -description
Gets or sets the UI element that will be clipped.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.SplitCloseThemeAnimation.OpenedTarget](/windows/winui/api/microsoft.ui.xaml.media.animation.splitclosethemeanimation.openedtarget) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The UI element that will be clipped.

## -remarks
Don't set this in XAML. For a XAML declaration, use [OpenedTargetName](splitclosethemeanimation_openedtargetname.md) instead.
