# Windows.UI.Xaml.Media.Animation.BasicConnectedAnimationConfiguration

## -description
An object that configures the connected animation as a linear transition from the source to the destination.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.BasicConnectedAnimationConfiguration](/windows/winui/api/microsoft.ui.xaml.media.animation.basicconnectedanimationconfiguration) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
