# Windows.UI.Xaml.Media.Animation.SplitCloseThemeAnimation.ContentTranslationOffset

## -description
Gets or sets the pixels to translate by when the animation runs.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.SplitCloseThemeAnimation.ContentTranslationOffset](/windows/winui/api/microsoft.ui.xaml.media.animation.splitclosethemeanimation.contenttranslationoffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -xaml-syntax
```xaml
<SplitCloseThemeAnimation ContentTranslationOffset="double" />
```


## -property-value
A distance in pixels.
