# Windows.UI.Xaml.Media.Animation.DrillInThemeAnimation.EntranceTargetName

## -description
Gets or sets the name of the element that is animating into view.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.DrillInThemeAnimation.EntranceTargetName](/windows/winui/api/microsoft.ui.xaml.media.animation.drillinthemeanimation.entrancetargetname) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The name of the element that is animating into view.
