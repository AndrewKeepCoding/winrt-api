# Windows.UI.Xaml.Media.Animation.DoubleKeyFrameCollection.Size

## -description
Gets the size (count) of the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.DoubleKeyFrameCollection.Size](/windows/winui/api/microsoft.ui.xaml.media.animation.doublekeyframecollection.size) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The count of items in the collection.
