# Windows.UI.Xaml.Media.Animation.EasingPointKeyFrame.EasingFunction

## -description
Gets or sets the easing function that is applied to the key frame.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.EasingPointKeyFrame.EasingFunction](/windows/winui/api/microsoft.ui.xaml.media.animation.easingpointkeyframe.easingfunction) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

  <EasingPointKeyFrame.EasingFunction>

</EasingPointKeyFrame>
