# Windows.UI.Xaml.Media.Animation

## -description

Provides animation and storyboard API for transition animations, visual states, or animated UI components.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation](/windows/winui/api/microsoft.ui.xaml.media.animation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
