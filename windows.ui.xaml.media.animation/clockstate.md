# ClockState

## -description
Describes the potential states of an animation.

Equivalent **WinUI 2 API for UWP**: [ClockState](/windows/winui/api/microsoft.ui.xaml.media.animation.clockstate) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -enum-fields
### -field Active:0
The current animation changes in direct relation to the animation of its parent.

### -field Filling:1
The animation continues and does not change in relation to the animation of its parent.

### -field Stopped:2
The animation is stopped.


## -remarks
This enumeration is not typically used in XAML.
