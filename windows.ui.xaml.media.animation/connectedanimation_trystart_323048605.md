# Windows.UI.Xaml.Media.Animation.ConnectedAnimation.TryStart

## -description
Attempts to start the animation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.ConnectedAnimation.TryStart](/windows/winui/api/microsoft.ui.xaml.media.animation.connectedanimation.trystart) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param destination
The element that is animated into view.

## -returns
**true** if the animation started; **false** if the animation has already been started, if it is no longer active, or if the destination element is the same as the source element.
