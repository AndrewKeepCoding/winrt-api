# Windows.UI.Xaml.Media.Animation.PointKeyFrameCollection.InsertAt

## -description
Inserts the specified item at the specified index.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.PointKeyFrameCollection.InsertAt](/windows/winui/api/microsoft.ui.xaml.media.animation.pointkeyframecollection.insertat) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param index
The index at which to insert the value.

### -param value
The value to insert.
