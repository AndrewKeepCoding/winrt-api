# Windows.UI.Xaml.Media.Animation.Storyboard.Resume

## -description
Resumes the animation clock, or run-time state, associated with the storyboard.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Animation.Storyboard.Resume](/windows/winui/api/microsoft.ui.xaml.media.animation.storyboard.resume) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
