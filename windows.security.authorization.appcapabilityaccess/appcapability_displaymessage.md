# Windows.Security.Authorization.AppCapabilityAccess.AppCapability.DisplayMessage

<!--
public string DisplayMessage { get; set; }
-->


## -description
The purpose of why the capability is accessed. 

## -property-value
The stored display message as a string property.

## -remarks
The display message, if specified, shows up in the system tray and settings app under Privacy & security > App permission pages while the capability is in use. Currently, only display messages of accesses to location, camera, microphone, contacts, phone calls, messaging, screenshots and apps show up in the settings app.
