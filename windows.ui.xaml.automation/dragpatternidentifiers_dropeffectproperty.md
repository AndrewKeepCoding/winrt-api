# Windows.UI.Xaml.Automation.DragPatternIdentifiers.DropEffectProperty

## -description
Gets the identifier for the [DropEffect](../windows.ui.xaml.automation.provider/idragprovider_dropeffect.md) automation property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.DragPatternIdentifiers.DropEffectProperty](/windows/winui/api/microsoft.ui.xaml.automation.dragpatternidentifiers.dropeffectproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
