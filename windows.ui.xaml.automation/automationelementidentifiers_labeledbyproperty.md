# Windows.UI.Xaml.Automation.AutomationElementIdentifiers.LabeledByProperty

## -description
Identifies the labeled-by peer automation property. The labeling relationship for an automation peer is returned by the [GetLabeledBy](../windows.ui.xaml.automation.peers/automationpeer_getlabeledby_1845689.md) method.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.LabeledByProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationelementidentifiers.labeledbyproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The automation property identifier.
