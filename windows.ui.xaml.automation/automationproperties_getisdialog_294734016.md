# Windows.UI.Xaml.Automation.AutomationProperties.GetIsDialog

## -description

Gets the value of the [AutomationProperties.IsDialog](automationproperties_isdialog.md) attached property for the specified [DependencyObject](../windows.ui.xaml/dependencyobject.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationProperties.GetIsDialog](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties.getisdialog) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters

### -param element

The object to check.

## -returns

**true** if the specified element is identified as a dialog window; otherwise, **false**.
