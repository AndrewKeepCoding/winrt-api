# Windows.UI.Xaml.Automation.AutomationProperties.GetHelpText

## -description
Gets the value of the [AutomationProperties.HelpText](automationproperties_helptext.md) attached property for the specified [DependencyObject](../windows.ui.xaml/dependencyobject.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationProperties.GetHelpText](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties.gethelptext) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param element
The [DependencyObject](../windows.ui.xaml/dependencyobject.md) to check.

## -returns
The help text for the specified element.
