# Windows.UI.Xaml.Automation.AutomationProperties.LevelProperty

## -description

Identifies the [AutomationProperties.Level](automationproperties_level.md) XAML attached property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationProperties.LevelProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationproperties.levelproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value

The identifier for the [AutomationProperties.Level](automationproperties_level.md) XAML attached property.
