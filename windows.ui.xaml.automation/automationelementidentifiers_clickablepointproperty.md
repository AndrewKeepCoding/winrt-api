# Windows.UI.Xaml.Automation.AutomationElementIdentifiers.ClickablePointProperty

## -description
Identifies the clickable point automation property. A valid clickable point property value is returned by the [GetClickablePoint](../windows.ui.xaml.automation.peers/automationpeer_getclickablepoint_955785073.md) method.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.ClickablePointProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationelementidentifiers.clickablepointproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The automation property identifier.
