# Windows.UI.Xaml.Automation.AutomationElementIdentifiers.ClassNameProperty

## -description
Identifies the class name automation property. The class name property value is returned by the [GetClassName](../windows.ui.xaml.automation.peers/automationpeer_getclassname_614238974.md) method.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.ClassNameProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationelementidentifiers.classnameproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The automation property identifier.
