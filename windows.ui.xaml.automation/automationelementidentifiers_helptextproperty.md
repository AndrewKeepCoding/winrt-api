# Windows.UI.Xaml.Automation.AutomationElementIdentifiers.HelpTextProperty

## -description
Identifies the help text automation property. The help text property value is returned by the [GetHelpText](../windows.ui.xaml.automation.peers/automationpeer_gethelptext_1251901404.md) method.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.HelpTextProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationelementidentifiers.helptextproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The automation property identifier.
