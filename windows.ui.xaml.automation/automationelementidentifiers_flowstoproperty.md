# Windows.UI.Xaml.Automation.AutomationElementIdentifiers.FlowsToProperty

## -description
Identifies the "flows to" automation property. The "flows to" property value is returned by the [GetFlowsTo](automationproperties_getflowsto_483856830.md) method.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Automation.AutomationElementIdentifiers.FlowsToProperty](/windows/winui/api/microsoft.ui.xaml.automation.automationelementidentifiers.flowstoproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The automation property identifier.
