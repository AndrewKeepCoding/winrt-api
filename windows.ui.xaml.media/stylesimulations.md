# StyleSimulations

## -description
Describes the simulation style of a font.

Equivalent **WinUI 2 API for UWP**: [StyleSimulations](/windows/winui/api/microsoft.ui.xaml.media.stylesimulations) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
