# Windows.UI.Xaml.Media.ScaleTransform.ScaleYProperty

## -description
Identifies the [ScaleY](scaletransform_scaley.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.ScaleTransform.ScaleYProperty](/windows/winui/api/microsoft.ui.xaml.media.scaletransform.scaleyproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ScaleY](scaletransform_scaley.md) dependency property.
