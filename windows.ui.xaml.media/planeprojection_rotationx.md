# Windows.UI.Xaml.Media.PlaneProjection.RotationX

## -description
Gets or sets the number of degrees to rotate the object around the x-axis of rotation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.PlaneProjection.RotationX](/windows/winui/api/microsoft.ui.xaml.media.planeprojection.rotationx) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
