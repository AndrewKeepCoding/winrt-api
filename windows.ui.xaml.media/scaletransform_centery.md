# Windows.UI.Xaml.Media.ScaleTransform.CenterY

## -description
Gets or sets the y-coordinate of the center point of this [ScaleTransform](scaletransform.md).

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.ScaleTransform.CenterY](/windows/winui/api/microsoft.ui.xaml.media.scaletransform.centery) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
