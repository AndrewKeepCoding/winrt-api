# Windows.UI.Xaml.Media.ScaleTransform.ScaleX

## -description
Gets or sets the x-axis scale factor.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.ScaleTransform.ScaleX](/windows/winui/api/microsoft.ui.xaml.media.scaletransform.scalex) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
