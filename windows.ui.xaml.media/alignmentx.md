# AlignmentX

## -description
Describes how content is positioned horizontally in a container.

Equivalent **WinUI 2 API for UWP**: [AlignmentX](/windows/winui/api/microsoft.ui.xaml.media.alignmentx) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

### -field Left:0

### -field Center:1

### -field Right:2
The contents align toward the right of the container.
