# Windows.UI.Xaml.Media.TransformGroup.ChildrenProperty

## -description

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.TransformGroup.ChildrenProperty](/windows/winui/api/microsoft.ui.xaml.media.transformgroup.childrenproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
