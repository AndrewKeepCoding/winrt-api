# Windows.UI.Xaml.Media.ScaleTransform.ScaleY

## -description
Gets or sets the y-axis scale factor.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.ScaleTransform.ScaleY](/windows/winui/api/microsoft.ui.xaml.media.scaletransform.scaley) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
