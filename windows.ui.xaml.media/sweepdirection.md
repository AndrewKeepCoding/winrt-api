# SweepDirection

## -description
Specifies the direction in which an elliptical arc is drawn.

Equivalent **WinUI 2 API for UWP**: [SweepDirection](/windows/winui/api/microsoft.ui.xaml.media.sweepdirection) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -enum-fields
### -field Counterclockwise:0
Arcs are drawn in a counterclockwise (negative-angle) direction.

### -field Clockwise:1
Arcs are drawn in a clockwise (positive-angle) direction.
