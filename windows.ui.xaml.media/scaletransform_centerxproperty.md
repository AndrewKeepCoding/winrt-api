# Windows.UI.Xaml.Media.ScaleTransform.CenterXProperty

## -description
Identifies the [CenterX](scaletransform_centerx.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.ScaleTransform.CenterXProperty](/windows/winui/api/microsoft.ui.xaml.media.scaletransform.centerxproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [CenterX](scaletransform_centerx.md) dependency property.
