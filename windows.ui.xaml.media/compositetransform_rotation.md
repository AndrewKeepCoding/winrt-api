# Windows.UI.Xaml.Media.CompositeTransform.Rotation

## -description
Gets or sets the angle, in degrees, of clockwise rotation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.CompositeTransform.Rotation](/windows/winui/api/microsoft.ui.xaml.media.compositetransform.rotation) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
