# Windows.UI.Xaml.Media.CompositeTransform.CenterYProperty

## -description
Identifies the [CenterY](compositetransform_centery.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.CompositeTransform.CenterYProperty](/windows/winui/api/microsoft.ui.xaml.media.compositetransform.centeryproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [CenterY](compositetransform_centery.md) dependency property.
