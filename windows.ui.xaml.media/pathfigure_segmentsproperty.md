# Windows.UI.Xaml.Media.PathFigure.SegmentsProperty

## -description
Identifies the [Segments](pathfigure_segments.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.PathFigure.SegmentsProperty](/windows/winui/api/microsoft.ui.xaml.media.pathfigure.segmentsproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Segments](pathfigure_segments.md) dependency property.
