# Windows.UI.Xaml.Media.TranslateTransform.YProperty

## -description
Identifies the [Y](translatetransform_y.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.TranslateTransform.YProperty](/windows/winui/api/microsoft.ui.xaml.media.translatetransform.yproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [Y](translatetransform_y.md) dependency property.
