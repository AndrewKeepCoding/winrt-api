# Windows.UI.Xaml.Media.GeometryCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.GeometryCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.media.geometrycollection.replaceall) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param items
The new collection items.

## -remarks
If successful, calling this method reduces the collection size by one.
