# Windows.UI.Xaml.Media.SkewTransform.AngleY

## -description
Gets or sets the y-axis skew angle, which is measured in degrees counterclockwise from the x-axis.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.SkewTransform.AngleY](/windows/winui/api/microsoft.ui.xaml.media.skewtransform.angley) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The skew angle, which is measured in degrees counterclockwise from the x-axis. The default is 0.
