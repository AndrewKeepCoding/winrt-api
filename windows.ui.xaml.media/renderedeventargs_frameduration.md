# Windows.UI.Xaml.Media.RenderedEventArgs.FrameDuration

## -description
Gets the duration of the time it took to render the most recent frame.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.RenderedEventArgs.FrameDuration](/windows/winui/api/microsoft.ui.xaml.media.renderedeventargs.frameduration) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The duration of the time it took to render the most recent frame.
