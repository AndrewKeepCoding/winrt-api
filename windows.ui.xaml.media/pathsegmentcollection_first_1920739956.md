# Windows.UI.Xaml.Media.PathSegmentCollection.First

## -description
Returns an iterator for the items in the collection.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.PathSegmentCollection.First](/windows/winui/api/microsoft.ui.xaml.media.pathsegmentcollection.first) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -returns
The iterator. The iterator's current position is the 0-index position, or at the collection end if the collection is empty.
