# Windows.UI.Xaml.Media.CompositeTransform.ScaleXProperty

## -description
Identifies the [ScaleX](compositetransform_scalex.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.CompositeTransform.ScaleXProperty](/windows/winui/api/microsoft.ui.xaml.media.compositetransform.scalexproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The identifier for the [ScaleX](compositetransform_scalex.md) dependency property.
