# Windows.UI.Xaml.Media.GeneralTransform.Inverse

## -description
Gets the inverse transformation of this [GeneralTransform](generaltransform.md), if possible.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.GeneralTransform.Inverse](/windows/winui/api/microsoft.ui.xaml.media.generaltransform.inverse) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
An inverse of this instance, if possible; otherwise null.
