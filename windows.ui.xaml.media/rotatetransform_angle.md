# Windows.UI.Xaml.Media.RotateTransform.Angle

## -description
Gets or sets the angle, in degrees, of clockwise rotation.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.RotateTransform.Angle](/windows/winui/api/microsoft.ui.xaml.media.rotatetransform.angle) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
