# Windows.UI.Xaml.Media.Brush.OpacityProperty

## -description
Identifies the [Opacity](brush_opacity.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Brush.OpacityProperty](/windows/winui/api/microsoft.ui.xaml.media.brush.opacityproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The [Opacity](brush_opacity.md) dependency property identifier.
