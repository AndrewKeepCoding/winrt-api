# Windows.UI.Xaml.Media.LinearGradientBrush.EndPoint

## -description
Gets or sets the ending two-dimensional coordinates of the linear gradient.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.LinearGradientBrush.EndPoint](/windows/winui/api/microsoft.ui.xaml.media.lineargradientbrush.endpoint) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

The ending two-dimensional coordinates of the linear gradient. The default is a [Point](../windows.foundation/point.md) with value 1,1.
