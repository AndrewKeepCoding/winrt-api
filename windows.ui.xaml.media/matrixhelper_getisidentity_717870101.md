# Windows.UI.Xaml.Media.MatrixHelper.GetIsIdentity

## -description
Returns whether the provided [Matrix](matrix.md) represents an **Identity** transform.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.MatrixHelper.GetIsIdentity](/windows/winui/api/microsoft.ui.xaml.media.matrixhelper.getisidentity) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -parameters
### -param target
The target [Matrix](matrix.md) to evaluate.

## -returns
**true** if the [Matrix](matrix.md) represents an **Identity** transform; otherwise, **false**.
