# Windows.UI.Xaml.Shapes.Shape.StrokeDashOffset

## -description
Gets or sets a value that specifies the distance within the dash pattern where a dash begins.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Shape.StrokeDashOffset](/windows/winui/api/microsoft.ui.xaml.shapes.shape.strokedashoffset) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
A value that represents the distance within the dash pattern where a dash begins. The default value is 0.
