# Windows.UI.Xaml.Shapes.Rectangle.RadiusY

## -description
Gets or sets the y-axis radius of the ellipse that is used to round the corners of the rectangle.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Rectangle.RadiusY](/windows/winui/api/microsoft.ui.xaml.shapes.rectangle.radiusy) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
The y-axis radius of the ellipse that is used to round the corners of the rectangle. The default is 0.
