# Windows.UI.Xaml.Shapes.Shape.StrokeProperty

## -description
Identifies the [Stroke](shape_stroke.md) dependency property.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Shape.StrokeProperty](/windows/winui/api/microsoft.ui.xaml.shapes.shape.strokeproperty) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
