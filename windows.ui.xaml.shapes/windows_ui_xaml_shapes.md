# Windows.UI.Xaml.Shapes

## -description
Defines basic shapes that are intended for decorative rendering or for compositing non-interactive parts of controls.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes](/windows/winui/api/microsoft.ui.xaml.shapes) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
