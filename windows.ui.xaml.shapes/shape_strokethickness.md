# Windows.UI.Xaml.Shapes.Shape.StrokeThickness

## -description
Gets or sets the width of the [Shape](shape.md) stroke outline.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Shape.StrokeThickness](/windows/winui/api/microsoft.ui.xaml.shapes.shape.strokethickness) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).

## -property-value
