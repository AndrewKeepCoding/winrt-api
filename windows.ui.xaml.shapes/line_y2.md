# Windows.UI.Xaml.Shapes.Line.Y2

## -description
Gets or sets the y-coordinate of the [Line](line.md) end point.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Line.Y2](/windows/winui/api/microsoft.ui.xaml.shapes.line.y2) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).


## -property-value
The y-coordinate for the end point of the line, in pixels. The default is 0.
