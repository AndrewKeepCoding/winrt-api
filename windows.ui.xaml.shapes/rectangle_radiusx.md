# Windows.UI.Xaml.Shapes.Rectangle.RadiusX

## -description
Gets or sets the x-axis radius of the ellipse that is used to round the corners of the rectangle.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Shapes.Rectangle.RadiusX](/windows/winui/api/microsoft.ui.xaml.shapes.rectangle.radiusx) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
