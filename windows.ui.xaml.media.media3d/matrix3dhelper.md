# Windows.UI.Xaml.Media.Media3D.Matrix3DHelper

## -description
Provides static utilities for [Matrix3D](matrix3d.md). C# and Microsoft Visual Basic code should use members of [Matrix3D](matrix3d.md) instead.

Equivalent **WinUI 2 API for UWP**: [Microsoft.UI.Xaml.Media.Media3D.Matrix3DHelper](/windows/winui/api/microsoft.ui.xaml.media.media3d.matrix3dhelper) (for WinUI in the [Windows App SDK](/windows/apps/windows-app-sdk/), see the **[Windows App SDK namespaces](/windows/windows-app-sdk/api/winrt/)**).
